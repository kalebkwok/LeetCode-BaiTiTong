## 一、今日模板回顾

### 虚拟头节点
```python
dummy = ListNode(0, head)
# ...
return dummy.next
```

### 链表翻转（三指针）
```python
pre, cur = None, head
while cur:
    nxt = cur.next
    cur.next = pre
    pre = cur
    cur = nxt
return pre
```

### 快慢指针（判环）
```python
slow = fast = head
while fast and fast.next:
    slow = slow.next
    fast = fast.next.next
    if slow == fast:
        return True
return False
```
