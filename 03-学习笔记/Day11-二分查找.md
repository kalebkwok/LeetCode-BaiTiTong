## 一、今日模板回顾

### 标准二分
```python
left, right = 0, len(nums) - 1
while left <= right:
    mid = left + (right - left) // 2
    if nums[mid] == target: return mid
    elif nums[mid] < target: left = mid + 1
    else: right = mid - 1
return -1
```

### 左边界二分
```python
# 找第一个 >= target 的位置
while left <= right:
    mid = left + (right - left) // 2
    if nums[mid] < target:
        left = mid + 1
    else:
        right = mid - 1
return left
```

### 右边界二分
```python
# 找最后一个 <= target 的位置
while left <= right:
    mid = left + (right - left) // 2
    if nums[mid] <= target:
        left = mid + 1
    else:
        right = mid - 1
return right
```
