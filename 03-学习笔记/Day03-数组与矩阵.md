## 一、今日模板回顾

### 三次翻转法
```python
nums.reverse()
nums[:k] = reversed(nums[:k])
nums[k:] = reversed(nums[k:])
```

### 前缀积+后缀积
```python
res[i] = prefix  # 先乘前缀
res[i] *= suffix  # 再乘后缀
```

### 螺旋矩阵边界收缩
