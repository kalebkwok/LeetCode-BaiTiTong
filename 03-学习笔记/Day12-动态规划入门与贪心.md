### DP 解题五步法
```
1. dp 数组含义
2. 状态转移方程
3. 初始化
4. 遍历顺序
5. 手动验证
```

### 背包问题框架
```python
# 0-1 背包：容量倒序遍历
for j in range(capacity, w - 1, -1):
    dp[j] = max(dp[j], dp[j - w] + v)

# 完全背包：容量正序遍历
for j in range(w, capacity + 1):
    dp[j] = max(dp[j], dp[j - w] + v)
```

### 贪心 vs DP
```python
# 贪心：每一步局部最优
# DP：记录所有可能，取最优
```
