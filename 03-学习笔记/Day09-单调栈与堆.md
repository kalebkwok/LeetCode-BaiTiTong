## 一、今日模板回顾

### 单调栈模板
```python
stack = []
for i in range(len(nums)):
    while stack and nums[stack[-1]] < nums[i]:
        idx = stack.pop()
        res[idx] = nums[i]
    stack.append(i)
```

### 堆操作
```python
import heapq
heapq.heappush(heap, val)   # 入堆
val = heapq.heappop(heap)    # 出堆
# 大根堆：存负值
heapq.heappush(heap, -val)
```
