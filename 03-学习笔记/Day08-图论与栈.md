```python
def dfs(i, j):
    if 越界 or grid[i][j] != '1':
        return
    grid[i][j] = '0'
    dfs(i+1, j); dfs(i-1, j); dfs(i, j+1); dfs(i, j-1)
```

### 拓扑排序（Kahn 算法）
```python
q = [i for i in range(n) if indegree[i] == 0]
while q:
    node = q.popleft()
    for neighbor in graph[node]:
        indegree[neighbor] -= 1
        if indegree[neighbor] == 0:
            q.append(neighbor)
```
