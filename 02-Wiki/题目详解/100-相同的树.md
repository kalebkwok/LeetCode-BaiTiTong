# 100. 相同的树 (Easy)

> **专题归类：** [[02-Wiki/专题总结/05-二叉树]]
> **LeetCode 链接：** https://leetcode.cn/problems/same-tree/

## 题目目标

判断两棵二叉树是否结构相同且对应节点值相等。

## 重点

- 同步递归比较两棵树的对应位置。
- `None` 的三种情形要先处理：都空、只有一边空、都非空。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n) |
| 空间复杂度 | O(h) |

## 解题思路

两个节点都为空则相同；仅一个为空或值不同则不同；否则继续比较它们的左右孩子。

## 代码实现

```python
class Solution:
    def isSameTree(self, p: TreeNode | None, q: TreeNode | None) -> bool:
        if not p and not q:
            return True
        if not p or not q or p.val != q.val:
            return False
        return self.isSameTree(p.left, q.left) and self.isSameTree(p.right, q.right)
```

## 易错点与复盘

- 只比较遍历值不够，结构不同也必须返回 False。
- `not p and not q` 必须放在“仅一边为空”的判断之前。
