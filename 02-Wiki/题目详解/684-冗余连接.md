# 684. 冗余连接 (Medium)

> **专题归类：** [[02-Wiki/专题总结/07-图论]]
> **LeetCode 链接：** https://leetcode.cn/problems/redundant-connection/

## 题目目标

无向图原本是树，多加一条边后形成环，返回最后导致环的边。

## 重点

- 并查集判断两端是否已在同一连通块。
- 加边前同根，说明这条边会形成环。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n α(n)) |
| 空间复杂度 | O(n) |

## 代码实现

```python
from typing import List


class Solution:
    def findRedundantConnection(self, edges: List[List[int]]) -> List[int]:
        parent = list(range(len(edges) + 1))
        def find(x):
            while x != parent[x]:
                parent[x] = parent[parent[x]]
                x = parent[x]
            return x
        for a, b in edges:
            root_a, root_b = find(a), find(b)
            if root_a == root_b:
                return [a, b]
            parent[root_a] = root_b
        return []
```

## 易错点与复盘

- `parent` 长度是边数加一，节点编号从 1 开始。
- 路径压缩只优化查找，不改变判环逻辑。

## 示例

```text
[[1,2],[1,3],[2,3]] 中加入 [2,3] 时两端已连通，返回它。
```
