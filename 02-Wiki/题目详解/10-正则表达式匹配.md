# 10. 正则表达式匹配 (Hard)

> **专题归类：** [[02-Wiki/专题总结/10-动态规划]]
> **LeetCode 链接：** https://leetcode.cn/problems/regular-expression-matching/

## 题目目标

实现只含 `.` 与 `*` 的正则匹配，要求覆盖整个字符串。

## 重点

- `dp[i][j]` 表示 `s[:i]` 是否匹配 `p[:j]`。
- `*` 可令前一字符出现 0 次，或在当前字符匹配时继续消费 `s`。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(mn) |
| 空间复杂度 | O(mn) |

## 代码实现

```python
class Solution:
    def isMatch(self, s: str, p: str) -> bool:
        dp = [[False] * (len(p) + 1) for _ in range(len(s) + 1)]
        dp[0][0] = True
        for j in range(2, len(p) + 1):
            if p[j - 1] == "*":
                dp[0][j] = dp[0][j - 2]
        for i in range(1, len(s) + 1):
            for j in range(1, len(p) + 1):
                if p[j - 1] != "*":
                    dp[i][j] = dp[i - 1][j - 1] and p[j - 1] in {s[i - 1], "."}
                else:
                    dp[i][j] = dp[i][j - 2]
                    if p[j - 2] in {s[i - 1], "."}:
                        dp[i][j] |= dp[i - 1][j]
        return dp[-1][-1]
```

## 易错点与复盘

- `*` 永远修饰前一个模式字符，因此使用 `j-2`。
- 空串可匹配 `a*b*c*`，需初始化第一行。

## 示例

```text
s="aab", p="c*a*b"：c* 取 0 次、a* 取 2 次，返回 True。
```
