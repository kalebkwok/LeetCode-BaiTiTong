# 567. 字符串的排列 (Medium)

> **专题归类：** [[02-Wiki/专题总结/02-双指针与滑动窗口]]
> **LeetCode 链接：** https://leetcode.cn/problems/permutation-in-string/

## 题目目标

判断 `s2` 是否包含 `s1` 任意排列形成的连续子串。

## 重点

- 排列要求字符频次相同，所以窗口长度固定为 `len(s1)`。
- 用 26 位计数数组比较频次。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n) |
| 空间复杂度 | O(1) |

## 代码实现

```python
class Solution:
    def checkInclusion(self, s1: str, s2: str) -> bool:
        if len(s1) > len(s2):
            return False
        need = [0] * 26
        window = [0] * 26
        for char in s1:
            need[ord(char) - ord("a")] += 1
        for i, char in enumerate(s2):
            window[ord(char) - ord("a")] += 1
            if i >= len(s1):
                window[ord(s2[i - len(s1)]) - ord("a")] -= 1
            if window == need:
                return True
        return False
```

## 易错点与复盘

- 加入新字符后，窗口超过固定长度才移出最左字符。
- 这题找的是连续子串，不是任意子序列。

## 示例

```text
s1="ab", s2="eidbaooo"，窗口 "ba" 频次匹配，返回 True。
```
