# 338. 比特位计数 (Easy)

> **专题归类：** [[02-Wiki/专题总结/12-技巧专题]]
> **LeetCode 链接：** https://leetcode.cn/problems/counting-bits/

## 题目目标

返回数组 `ans`，其中 `ans[i]` 表示 `0 <= i <= n` 中 `i` 的二进制 `1` 的个数。

## 重点

- `i & (i - 1)` 比 `i` 少一个最低位 `1`。
- 递推式：`ans[i] = ans[i & (i - 1)] + 1`。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n) |
| 空间复杂度 | O(n) |

## 解题思路

从小到大构造答案。对每个 `i`，先找到删去最低位 `1` 后的更小数；它的答案已知，只需再加 1。

## 代码实现

```python
from typing import List


class Solution:
    def countBits(self, n: int) -> List[int]:
        ans = [0] * (n + 1)
        for i in range(1, n + 1):
            ans[i] = ans[i & (i - 1)] + 1
        return ans
```

## 易错点与复盘

- `ans` 长度是 `n + 1`，因为要包含 0。
- 递推依赖的下标严格小于 `i`，所以可以从左到右计算。
