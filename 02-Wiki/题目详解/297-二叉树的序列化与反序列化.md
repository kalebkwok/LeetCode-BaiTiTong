# 297. 二叉树的序列化与反序列化 (Hard)

> **专题归类：** [[02-Wiki/专题总结/05-二叉树]]
> **LeetCode 链接：** https://leetcode.cn/problems/serialize-and-deserialize-binary-tree/

## 题目目标

把二叉树编码成字符串，并能从该字符串还原出完全相同的树。

## 重点

- 前序遍历必须记录空节点，否则无法恢复树形结构。
- 反序列化时按同样的前序顺序消费 token。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n) |
| 空间复杂度 | O(n) |

## 解题思路

序列化中用 `#` 表示空节点，节点值用逗号分隔。反序列化用迭代器逐个读取：读到 `#` 返回空，否则创建节点并递归建左、右子树。

## 代码实现

```python
class Codec:
    def serialize(self, root: TreeNode | None) -> str:
        values = []

        def dfs(node: TreeNode | None) -> None:
            if not node:
                values.append("#")
                return
            values.append(str(node.val))
            dfs(node.left)
            dfs(node.right)

        dfs(root)
        return ",".join(values)

    def deserialize(self, data: str) -> TreeNode | None:
        values = iter(data.split(","))

        def build() -> TreeNode | None:
            value = next(values)
            if value == "#":
                return None
            node = TreeNode(int(value))
            node.left = build()
            node.right = build()
            return node

        return build()
```

## 易错点与复盘

- 只保存非空节点值会丢失结构，例如左右孩子缺失的位置无法区分。
- 序列化和反序列化的遍历顺序必须完全一致。
