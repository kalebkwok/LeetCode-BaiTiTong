# 133. 克隆图 (Medium)

> **专题归类：** [[02-Wiki/专题总结/07-图论]]
> **LeetCode 链接：** https://leetcode.cn/problems/clone-graph/

## 题目目标

深拷贝一个无向连通图。

## 重点

- 哈希表把原节点映射到克隆节点。
- 创建克隆节点后立刻缓存，才能处理环。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(V+E) |
| 空间复杂度 | O(V) |

## 代码实现

```python
class Solution:
    def cloneGraph(self, node: 'Node | None') -> 'Node | None':
        if not node:
            return None
        copies = {}
        def dfs(current):
            if current in copies:
                return copies[current]
            copy = Node(current.val)
            copies[current] = copy
            copy.neighbors = [dfs(neighbor) for neighbor in current.neighbors]
            return copy
        return dfs(node)
```

## 易错点与复盘

- 缓存一定在递归邻居之前写入，否则环会无限递归。
- 不能复用原节点的邻居列表。

## 示例

```text
1 与 2 互相相连时，克隆后新 1、新 2 也互连，但都不是原对象。
```
