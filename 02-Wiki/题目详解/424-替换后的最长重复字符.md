# 424. 替换后的最长重复字符 (Medium)

> **专题归类：** [[02-Wiki/专题总结/02-双指针与滑动窗口]]
> **LeetCode 链接：** https://leetcode.cn/problems/longest-repeating-character-replacement/

## 题目目标

最多替换 `k` 个字符后，求可变为同一字符的最长子串长度。

## 重点

- 窗口需要替换数为 `窗口长度 - 窗口内最高频数`。
- 当需要替换数大于 `k` 时，左边界收缩。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n) |
| 空间复杂度 | O(1) |

大写字母字符集大小固定。

## 代码实现

```python
from collections import defaultdict


class Solution:
    def characterReplacement(self, s: str, k: int) -> int:
        count = defaultdict(int)
        left = max_count = 0
        for right, char in enumerate(s):
            count[char] += 1
            max_count = max(max_count, count[char])
            while right - left + 1 - max_count > k:
                count[s[left]] -= 1
                left += 1
        return len(s) - left
```

## 易错点与复盘

- `max_count` 不必在左移时降低；它作为历史上界仍能保证答案正确。
- 不要把“最多保留的字符”误认为当前窗口唯一字符。

## 示例

```text
s="AABABBA", k=1，最长可统一为 "A" 的窗口长度是 4。
```
