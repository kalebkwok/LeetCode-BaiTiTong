# 134. 加油站 (Medium)

> **专题归类：** [[02-Wiki/专题总结/11-贪心算法]]
> **LeetCode 链接：** https://leetcode.cn/problems/gas-station/

## 题目目标

在环路中找一个起点，使车能绕行一周；不存在则返回 -1。

## 重点

- 总油量小于总消耗时，无解。
- 从起点到当前位置油量为负，期间任一点都不能当起点，直接从下一站重启。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n) |
| 空间复杂度 | O(1) |

## 代码实现

```python
from typing import List


class Solution:
    def canCompleteCircuit(self, gas: List[int], cost: List[int]) -> int:
        if sum(gas) < sum(cost):
            return -1
        start = tank = 0
        for i in range(len(gas)):
            tank += gas[i] - cost[i]
            if tank < 0:
                start = i + 1
                tank = 0
        return start
```

## 易错点与复盘

- 总量条件只说明是否有解；局部负数时仍需移动起点。
- 返回的是站点下标，不是当前剩余油量。

## 示例

```text
gas=[1,2,3,4,5], cost=[3,4,5,1,2]，从 3 出发可绕行一周。
```
