# 261. 以图判树 (Medium)

> **专题归类：** [[02-Wiki/专题总结/07-图论]]
> **LeetCode 链接：** https://leetcode.ca/all/261.html

## 题目目标

判断无向图是否是一棵合法树。

## 重点

- `n` 个节点的树必须有 `n-1` 条边。
- 边数正确后，只需从 0 DFS 验证所有节点连通。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(V+E) |
| 空间复杂度 | O(V+E) |

## 代码实现

```python
from collections import defaultdict
from typing import List


class Solution:
    def validTree(self, n: int, edges: List[List[int]]) -> bool:
        if len(edges) != n - 1:
            return False
        graph = defaultdict(list)
        for a, b in edges:
            graph[a].append(b); graph[b].append(a)
        seen, stack = {0}, [0]
        while stack:
            node = stack.pop()
            for neighbor in graph[node]:
                if neighbor not in seen:
                    seen.add(neighbor); stack.append(neighbor)
        return len(seen) == n
```

## 易错点与复盘

- 边数 `n-1` 加连通性等价于无环且连通。
- `n=1`、无边时也是树。

## 示例

```text
n=5，4 条边且全部节点连通，返回 True。
```
