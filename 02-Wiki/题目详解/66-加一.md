# 66. 加一 (Easy)

> **专题归类：** [[02-Wiki/专题总结/03-数组与矩阵]]
> **LeetCode 链接：** https://leetcode.cn/problems/plus-one/

## 题目目标

用数字数组表示非负整数，返回该整数加一后的数组。

## 重点

- 从最低位向最高位模拟进位。
- 遇到小于 9 的位加一即可结束；全为 9 时需要新首位。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n) |
| 空间复杂度 | O(1) |

除全为 9 时返回的新数组外，原地修改。

## 代码实现

```python
from typing import List


class Solution:
    def plusOne(self, digits: List[int]) -> List[int]:
        for i in range(len(digits) - 1, -1, -1):
            if digits[i] < 9:
                digits[i] += 1
                return digits
            digits[i] = 0
        return [1] + digits
```

## 易错点与复盘

- `[9,9]` 不是 `[0,0]`，而是 `[1,0,0]`。
- 一旦某位没有进位，立刻返回，前面高位无需处理。

## 示例

```text
[1,2,9] 从末位进位后得到 [1,3,0]。
```
