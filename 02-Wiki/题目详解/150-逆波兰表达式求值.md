# 150. 逆波兰表达式求值 (Medium)

> **专题归类：** [[02-Wiki/专题总结/06-栈与堆]]
> **LeetCode 链接：** https://leetcode.cn/problems/evaluate-reverse-polish-notation/

---

## 题目目标

给定一个逆波兰表达式 `tokens`，计算表达式的值。

逆波兰表达式的特点是：操作符出现在操作数后面，例如普通表达式 `(2 + 1) * 3` 写成 `2 1 + 3 *`。

---

## 重点

- **遇到数字就入栈，遇到运算符就弹出两个数字计算。**
- 弹出顺序很重要：先弹出的是右操作数 `b`，后弹出的是左操作数 `a`。
- 除法要向 0 截断，Python 中可以用 `int(a / b)`。

---

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n) |
| 空间复杂度 | O(n) |

`n` 是 token 数量。每个 token 最多入栈/出栈一次。

---

## 解题思路

逆波兰表达式天然适合用栈。

从左到右扫描：

1. 如果当前 token 是数字，压入栈。
2. 如果当前 token 是运算符，弹出两个数。
3. 用这两个数计算结果，再把结果压回栈。
4. 扫描结束后，栈里唯一的数就是答案。

关键是顺序：

```text
栈里先有 a，再有 b
遇到操作符 op
计算的是 a op b
```

所以代码里要写成：

```python
b = stack.pop()
a = stack.pop()
```

---

## 代码实现

```python
from typing import List


class Solution:
    def evalRPN(self, tokens: List[str]) -> int:
        stack = []

        for token in tokens:
            if token not in {"+", "-", "*", "/"}:
                stack.append(int(token))
                continue

            b = stack.pop()
            a = stack.pop()

            if token == "+":
                stack.append(a + b)
            elif token == "-":
                stack.append(a - b)
            elif token == "*":
                stack.append(a * b)
            else:
                stack.append(int(a / b))  # 向 0 截断

        return stack[-1]
```

---

## 易错点与复盘

- **减法和除法不能写反：** `a - b` 和 `a / b`，不是 `b - a`。
- **除法不是地板除：** Python 的 `//` 是向下取整，负数时会错；这里要向 0 截断。
- **数字 token 可能是负数：** 不要用 `token.isdigit()` 判断数字，因为 `"-11".isdigit()` 是 `False`。
