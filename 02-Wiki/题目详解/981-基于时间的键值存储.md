# 981. 基于时间的键值存储 (Medium)

> **专题归类：** [[02-Wiki/专题总结/09-二分查找]]
> **LeetCode 链接：** https://leetcode.cn/problems/time-based-key-value-store/

## 题目目标

实现按时间戳存取值：查询时返回不晚于指定时间戳的最新值。

## 重点

- 每个 `key` 对应按时间递增的 `(timestamp, value)` 列表。
- 查询在该列表中找最后一个 `timestamp <= query_time` 的位置。

## 时间/空间复杂度

| 操作 | 复杂度 |
|---|---|
| `set` | O(1) |
| `get` | O(log n) |
| 空间 | O(n) |

## 解题思路

题目保证同一键的写入时间严格递增，所以直接追加。查询用二分维护候选答案 `answer`；每遇到不晚于目标的时间就记录它并向右找更新值。

## 代码实现

```python
from collections import defaultdict


class TimeMap:
    def __init__(self):
        self.data = defaultdict(list)

    def set(self, key: str, value: str, timestamp: int) -> None:
        self.data[key].append((timestamp, value))

    def get(self, key: str, timestamp: int) -> str:
        pairs = self.data[key]
        left, right, answer = 0, len(pairs) - 1, ""
        while left <= right:
            mid = (left + right) // 2
            if pairs[mid][0] <= timestamp:
                answer = pairs[mid][1]
                left = mid + 1
            else:
                right = mid - 1
        return answer
```

## 易错点与复盘

- 查询的是“最近且不超过”，不是第一个大于等于时间戳的值。
- 没有历史值时返回空字符串。
