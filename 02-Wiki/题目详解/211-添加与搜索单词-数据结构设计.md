# 211. 添加与搜索单词 - 数据结构设计 (Medium)

> **专题归类：** [[02-Wiki/专题总结/07-图论]]
> **LeetCode 链接：** https://leetcode.cn/problems/design-add-and-search-words-data-structure/

## 题目目标

设计单词字典，支持添加单词及包含 `.` 通配符的查询。

## 重点

- Trie 的普通字符沿唯一孩子走；`.` 需要尝试所有孩子。
- 通配符搜索本质是 Trie 上的 DFS。

## 时间/空间复杂度

| 操作 | 复杂度 |
|---|---|
| `addWord` | O(L) |
| `search` | 最坏 O(26^L) |
| 空间 | O(总字符数) |

## 代码实现

```python
class TrieNode:
    def __init__(self):
        self.children = {}
        self.word = False


class WordDictionary:
    def __init__(self):
        self.root = TrieNode()

    def addWord(self, word: str) -> None:
        node = self.root
        for char in word:
            node = node.children.setdefault(char, TrieNode())
        node.word = True

    def search(self, word: str) -> bool:
        def dfs(i: int, node: TrieNode) -> bool:
            if i == len(word):
                return node.word
            if word[i] == ".":
                return any(dfs(i + 1, child) for child in node.children.values())
            return word[i] in node.children and dfs(i + 1, node.children[word[i]])
        return dfs(0, self.root)
```

## 易错点与复盘

- 词尾标记不能省略，`bad` 不能因为存在 `badly` 就匹配。
- `.` 不能只选一个孩子，必须分支搜索。

## 示例

```text
加入 bad、dad 后，search(".ad") 返回 True。
```
