# 787. K 站中转内最便宜的航班 (Medium)

> **专题归类：** [[02-Wiki/专题总结/07-图论]] · [[02-Wiki/专题总结/10-动态规划]]
> **LeetCode 链接：** https://leetcode.cn/problems/cheapest-flights-within-k-stops/

## 题目目标

在最多 `k` 个中转站内，从 `src` 到 `dst` 的最便宜航班价格。

## 重点

- `k` 个中转最多使用 `k+1` 条边。
- Bellman-Ford 每轮只基于上一轮价格，防止同轮用超过限制的边。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O((k+1)E) |
| 空间复杂度 | O(V) |

## 代码实现

```python
from typing import List


class Solution:
    def findCheapestPrice(self, n: int, flights: List[List[int]], src: int, dst: int, k: int) -> int:
        prices = [float("inf")] * n
        prices[src] = 0
        for _ in range(k + 1):
            next_prices = prices[:]
            for source, destination, price in flights:
                if prices[source] != float("inf"):
                    next_prices[destination] = min(next_prices[destination], prices[source] + price)
            prices = next_prices
        return -1 if prices[dst] == float("inf") else prices[dst]
```

## 易错点与复盘

- 必须复制 `prices`；直接原地更新会在同一轮连续使用多条边。
- “k 站中转”不是 k 条边，而是 k+1 条航班。

## 示例

```text
0→1→2，k=1 时允许两条边，若价格 100+100，答案为 200。
```
