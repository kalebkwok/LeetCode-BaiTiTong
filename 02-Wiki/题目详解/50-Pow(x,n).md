# 50. Pow(x, n) (Medium)

> **专题归类：** [[02-Wiki/专题总结/12-技巧专题]]
> **LeetCode 链接：** https://leetcode.cn/problems/powx-n/

## 题目目标

计算 `x` 的整数次幂 `x^n`，其中 `n` 可以为负数。

## 重点

- 快速幂把指数每次减半，而不是逐次乘 `n` 次。
- 指数的当前最低位为 1 时，把当前底数乘进答案。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(log |n|) |
| 空间复杂度 | O(1) |

## 解题思路

若 `n < 0`，先把问题改写成 `(1 / x) ^ (-n)`。之后不断检查 `n` 的奇偶：奇数把 `base` 乘到 `result`，每轮把 `base` 平方、`n` 整除 2。

## 代码实现

```python
class Solution:
    def myPow(self, x: float, n: int) -> float:
        if n < 0:
            x = 1 / x
            n = -n

        result = 1.0
        while n:
            if n & 1:
                result *= x
            x *= x
            n >>= 1

        return result
```

## 易错点与复盘

- 负指数先取倒数，再把指数转正。
- `n & 1` 判断当前二进制最低位，不是判断原始指数是否只出现一次。
