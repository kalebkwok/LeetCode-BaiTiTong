# 647. 回文子串 (Medium)

> **专题归类：** [[02-Wiki/专题总结/10-动态规划]]
> **LeetCode 链接：** https://leetcode.cn/problems/palindromic-substrings/

## 题目目标

统计字符串中回文子串的数量。

## 重点

- 每个回文都有一个奇数中心或偶数中心。
- 从中心向两侧扩展，匹配一次就计一个回文。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n²) |
| 空间复杂度 | O(1) |

## 代码实现

```python
class Solution:
    def countSubstrings(self, s: str) -> int:
        total = 0
        for center in range(2 * len(s) - 1):
            left, right = center // 2, center // 2 + center % 2
            while left >= 0 and right < len(s) and s[left] == s[right]:
                total += 1
                left -= 1
                right += 1
        return total
```

## 易错点与复盘

- `2*n-1` 个中心同时覆盖单字符中心和字符间中心。
- 每次扩展成功的当前区间都是一个新回文。

## 示例

```text
"aaa" 的回文子串为 a、a、a、aa、aa、aaa，共 6 个。
```
