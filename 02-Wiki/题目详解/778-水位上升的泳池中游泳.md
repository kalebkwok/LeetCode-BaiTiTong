# 778. 水位上升的泳池中游泳 (Hard)

> **专题归类：** [[02-Wiki/专题总结/07-图论]]
> **LeetCode 链接：** https://leetcode.cn/problems/swim-in-rising-water/

## 题目目标

水位随时间上升，求从左上到右下所需的最小时间。

## 重点

- 路径代价是经过格子的最大高度，使用最小堆优先扩展当前最低可达水位。
- 首次弹出终点时的高度就是最优答案。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n² log n) |
| 空间复杂度 | O(n²) |

## 代码实现

```python
import heapq
from typing import List


class Solution:
    def swimInWater(self, grid: List[List[int]]) -> int:
        n = len(grid)
        heap, seen = [(grid[0][0], 0, 0)], {(0, 0)}
        while heap:
            time, r, c = heapq.heappop(heap)
            if (r, c) == (n - 1, n - 1):
                return time
            for dr, dc in ((1,0),(-1,0),(0,1),(0,-1)):
                nr, nc = r + dr, c + dc
                if 0 <= nr < n and 0 <= nc < n and (nr, nc) not in seen:
                    seen.add((nr, nc))
                    heapq.heappush(heap, (max(time, grid[nr][nc]), nr, nc))
        return -1
```

## 易错点与复盘

- 堆键是整条路径目前的最大高度，不是下一格单独高度。
- 节点入堆即标记，避免重复扩展。

## 示例

```text
[[0,2],[1,3]] 要等到水位 3 才能抵达右下角。
```
