# 1448. 统计二叉树中好节点的数目 (Medium)

> **专题归类：** [[02-Wiki/专题总结/05-二叉树]]
> **LeetCode 链接：** https://leetcode.cn/problems/count-good-nodes-in-binary-tree/

## 题目目标

统计从根到该节点的路径上，没有比它更大节点的“好节点”数量。

## 重点

- DFS 参数 `path_max` 保存当前路径最大值。
- 节点值 `>= path_max` 时是好节点。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n) |
| 空间复杂度 | O(h) |

## 解题思路

访问节点时先与路径最大值比较，再把 `max(path_max, node.val)` 传给两棵子树。不同分支各自带着自己的路径状态。

## 代码实现

```python
class Solution:
    def goodNodes(self, root: TreeNode) -> int:
        def dfs(node: TreeNode | None, path_max: int) -> int:
            if not node:
                return 0
            good = int(node.val >= path_max)
            next_max = max(path_max, node.val)
            return good + dfs(node.left, next_max) + dfs(node.right, next_max)

        return dfs(root, root.val)
```

## 易错点与复盘

- 相等也算好节点，所以比较符号是 `>=`。
- `path_max` 是递归参数，不需要在回溯时手动恢复。

```text

```
