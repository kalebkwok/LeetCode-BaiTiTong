# 875. 爱吃香蕉的珂珂 (Medium)

> **专题归类：** [[02-Wiki/专题总结/09-二分查找]]
> **LeetCode 链接：** https://leetcode.cn/problems/koko-eating-bananas/

## 题目目标

在 `h` 小时内吃完所有香蕉，求最小整数吃香蕉速度。

## 重点

- 二分的是答案速度 `k`，不是数组下标。
- 速度越大，总耗时单调不增，可行性具有单调性。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n log M) |
| 空间复杂度 | O(1) |

`M` 是最大堆大小。

## 解题思路

速度范围是 `[1, max(piles)]`。对中点速度计算总小时数；可行则保留左半寻找更小速度，不可行则加速。

## 代码实现

```python
from typing import List


class Solution:
    def minEatingSpeed(self, piles: List[int], h: int) -> int:
        left, right = 1, max(piles)
        while left < right:
            speed = (left + right) // 2
            hours = sum((pile + speed - 1) // speed for pile in piles)
            if hours <= h:
                right = speed
            else:
                left = speed + 1
        return left
```

## 易错点与复盘

- 向上取整用 `(pile + speed - 1) // speed`，不能直接整除。
- 可行时收缩右边界到 `speed`，因为它仍可能是最小答案。
