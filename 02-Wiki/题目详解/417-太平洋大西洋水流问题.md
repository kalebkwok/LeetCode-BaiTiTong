# 417. 太平洋大西洋水流问题 (Medium)

> **专题归类：** [[02-Wiki/专题总结/07-图论]]
> **LeetCode 链接：** https://leetcode.cn/problems/pacific-atlantic-water-flow/

## 题目目标

找出既能流向太平洋又能流向大西洋的格子。

## 重点

- 反向思考：从两片海岸向高处 DFS，而不是从每个格子向低处试流。
- 两次可达集合的交集即答案。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(mn) |
| 空间复杂度 | O(mn) |

## 代码实现

```python
from typing import List


class Solution:
    def pacificAtlantic(self, heights: List[List[int]]) -> List[List[int]]:
        rows, cols = len(heights), len(heights[0])
        pacific, atlantic = set(), set()
        def dfs(r, c, seen):
            seen.add((r, c))
            for dr, dc in ((1,0),(-1,0),(0,1),(0,-1)):
                nr, nc = r + dr, c + dc
                if 0 <= nr < rows and 0 <= nc < cols and (nr, nc) not in seen and heights[nr][nc] >= heights[r][c]:
                    dfs(nr, nc, seen)
        for r in range(rows):
            dfs(r, 0, pacific); dfs(r, cols - 1, atlantic)
        for c in range(cols):
            dfs(0, c, pacific); dfs(rows - 1, c, atlantic)
        return [[r, c] for r in range(rows) for c in range(cols) if (r, c) in pacific and (r, c) in atlantic]
```

## 易错点与复盘

- 反向 DFS 的高度条件是 `next >= current`。
- 两海的边界都要作为起点。

## 示例

```text
同时能反向到达两侧边界的格子才输出。
```
