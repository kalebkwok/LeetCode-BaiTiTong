# 253. 会议室 II (Medium)

> **专题归类：** [[02-Wiki/专题总结/03-数组与矩阵]] · [[02-Wiki/专题总结/06-栈与堆]]
> **LeetCode 链接：** https://leetcode.ca/all/253.html

## 题目目标

返回安排所有会议所需的最少会议室数量。

## 重点

- 小顶堆存正在使用的会议室结束时间。
- 当前会议开始前结束最早的会议室可复用。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n log n) |
| 空间复杂度 | O(n) |

## 代码实现

```python
import heapq
from typing import List


class Solution:
    def minMeetingRooms(self, intervals: List[List[int]]) -> int:
        intervals.sort(key=lambda interval: interval[0])
        rooms = []
        for start, end in intervals:
            if rooms and rooms[0] <= start:
                heapq.heappop(rooms)
            heapq.heappush(rooms, end)
        return len(rooms)
```

## 易错点与复盘

- 只需弹出一个结束时间：按开始时间逐个处理，新增会议一次只占用一个房间。
- 判断可复用用 `<= start`。

## 示例

```text
[[0,30],[5,10],[15,20]] 的最大同时会议数为 2。
```
