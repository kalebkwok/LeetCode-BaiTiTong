# 1899. 合并若干三元组以形成目标三元组 (Medium)

> **专题归类：** [[02-Wiki/专题总结/11-贪心算法]]
> **LeetCode 链接：** https://leetcode.cn/problems/merge-triplets-to-form-target-triplet/

## 题目目标

每次取三个三元组的逐维最大值，判断能否形成 `target`。

## 重点

- 任一维超过目标的三元组永远不能使用。
- 在有效三元组中，分别找到能提供 target 三个维度的值即可。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n) |
| 空间复杂度 | O(1) |

## 代码实现

```python
from typing import List


class Solution:
    def mergeTriplets(self, triplets: List[List[int]], target: List[int]) -> bool:
        matched = [False, False, False]
        for triplet in triplets:
            if any(triplet[i] > target[i] for i in range(3)):
                continue
            for i in range(3):
                if triplet[i] == target[i]:
                    matched[i] = True
        return all(matched)
```

## 易错点与复盘

- 不能使用任一维超标的三元组，即使它的其他维度有用。
- 不必真的模拟全部合并，只需追踪三个目标维度能否分别达成。

## 示例

```text
[[2,5,3],[1,8,4],[1,7,5]] 可形成 target=[2,7,5]。
```
