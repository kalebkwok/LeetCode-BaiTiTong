# 678. 有效的括号字符串 (Medium)

> **专题归类：** [[02-Wiki/专题总结/11-贪心算法]]
> **LeetCode 链接：** https://leetcode.cn/problems/valid-parenthesis-string/

## 题目目标

字符串只含 `(`、`)`、`*`；`*` 可代表左括号、右括号或空串，判断能否组成有效括号串。

## 重点

- `low` / `high` 是当前可能的未匹配左括号数量区间。
- `*` 使 `low` 最多减一、`high` 加一。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n) |
| 空间复杂度 | O(1) |

## 代码实现

```python
class Solution:
    def checkValidString(self, s: str) -> bool:
        low = high = 0
        for char in s:
            if char == "(":
                low += 1
                high += 1
            elif char == ")":
                low -= 1
                high -= 1
            else:
                low -= 1
                high += 1
            if high < 0:
                return False
            low = max(low, 0)
        return low == 0
```

## 易错点与复盘

- `high < 0` 说明即使所有 `*` 当左括号也不够，立即失败。
- `low` 不能小于 0，因为可把部分 `*` 当空串或左括号。

## 示例

```text
"(*)" 可令 * 为空，得到 "()"，返回 True。
```
