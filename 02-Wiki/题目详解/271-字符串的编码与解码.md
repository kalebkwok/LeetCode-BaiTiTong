# 271. 字符串的编码与解码 (Medium)

> **专题归类：** [[02-Wiki/专题总结/01-哈希表]]
> **LeetCode 链接：** https://leetcode.ca/all/271.html

## 题目目标

设计可逆编码，把字符串列表编码成单一字符串并精确解码。

## 重点

- 分隔符方案会与原字符串内容冲突。
- 使用 `长度#内容` 前缀，先读长度再读对应字符数。

## 时间/空间复杂度

| 操作 | 复杂度 |
|---|---|
| `encode` | O(总字符数) |
| `decode` | O(总字符数) |
| 空间 | O(总字符数) |

## 代码实现

```python
from typing import List


class Codec:
    def encode(self, strs: List[str]) -> str:
        return "".join(f"{len(s)}#{s}" for s in strs)

    def decode(self, s: str) -> List[str]:
        result, i = [], 0
        while i < len(s):
            j = s.index("#", i)
            size = int(s[i:j])
            i = j + 1
            result.append(s[i:i + size])
            i += size
        return result
```

## 易错点与复盘

- 空字符串编码为 `0#`，仍能正确还原。
- 不能按 `#` 直接 `split`，因为原内容本身可能有 `#`。

## 示例

```text
["lint","code","#"] 编码为 "4#lint4#code1##"。
```
