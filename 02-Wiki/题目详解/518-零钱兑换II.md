# 518. 零钱兑换 II (Medium)

> **专题归类：** [[02-Wiki/专题总结/10-动态规划]]
> **LeetCode 链接：** https://leetcode.cn/problems/coin-change-ii/

## 题目目标

给定硬币面额，求凑出 `amount` 的组合数；硬币可无限使用，顺序不同不重复计数。

## 重点

- `dp[x]` 表示凑成金额 `x` 的组合数。
- 外层遍历硬币、内层正序遍历金额，才能避免把排列重复计数。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(amount × n) |
| 空间复杂度 | O(amount) |

## 代码实现

```python
from typing import List


class Solution:
    def change(self, amount: int, coins: List[int]) -> int:
        dp = [0] * (amount + 1)
        dp[0] = 1
        for coin in coins:
            for value in range(coin, amount + 1):
                dp[value] += dp[value - coin]
        return dp[amount]
```

## 易错点与复盘

- `dp[0] = 1` 表示什么都不选也是一种构造方式。
- 内层必须正序，才能允许同一硬币重复使用。

## 示例

```text
amount=5, coins=[1,2,5]，组合为 5、2+2+1、2+1+1+1、1×5，共 4 种。
```
