# 572. 另一棵树的子树 (Easy)

> **专题归类：** [[02-Wiki/专题总结/05-二叉树]]
> **LeetCode 链接：** https://leetcode.cn/problems/subtree-of-another-tree/

## 题目目标

判断 `subRoot` 是否与 `root` 中某个节点为根的完整子树相同。

## 重点

- 外层 DFS 枚举 `root` 的每个候选根。
- 内层 `same` 精确比较两棵树的结构和值。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(mn) |
| 空间复杂度 | O(m+n) |

## 解题思路

在每个 `root` 节点处尝试 `same(root, subRoot)`；不相同就继续去左右子树寻找候选根。

## 代码实现

```python
class Solution:
    def isSubtree(self, root: TreeNode | None, subRoot: TreeNode | None) -> bool:
        def same(a: TreeNode | None, b: TreeNode | None) -> bool:
            if not a and not b:
                return True
            if not a or not b or a.val != b.val:
                return False
            return same(a.left, b.left) and same(a.right, b.right)

        if not root:
            return False
        return same(root, subRoot) or self.isSubtree(root.left, subRoot) or self.isSubtree(root.right, subRoot)
```

## 易错点与复盘

- “子树”要求整个结构一致，不能只找到相同值节点。
- 题目保证 `subRoot` 非空；若自行扩展，空树可定义为任意树的子树。
