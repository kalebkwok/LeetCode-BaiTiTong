# 90. 子集 II (Medium)

> **专题归类：** [[02-Wiki/专题总结/08-回溯算法]]
> **LeetCode 链接：** https://leetcode.cn/problems/subsets-ii/

---

## 题目目标

给定一个可能包含重复元素的整数数组 `nums`，返回所有可能的子集。

结果中不能包含重复子集。

---

## 重点

- 这是 [[02-Wiki/题目详解/78-子集|78-子集]] 的进阶版：输入里有重复元素。
- 子集问题每个递归节点都是答案，所以进入递归后立刻记录 `path`。
- 排序后用 `start` 控制只向后选，用同层去重跳过重复分支。
- 去重条件仍然是：`if i > start and nums[i] == nums[i - 1]: continue`。

---

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n · 2^n) |
| 空间复杂度 | O(n) |

不计输出结果空间。总子集数量最多 2^n，每个子集拷贝最多 O(n)。

---

## 解题思路

普通子集题里，数组元素互不相同，所以只需要：

```python
res.append(path[:])
for i in range(start, len(nums)):
    path.append(nums[i])
    backtrack(i + 1)
    path.pop()
```

但这题有重复元素，例如：

```text
[1, 2, 2]
```

如果不去重，会从第一个 `2` 和第二个 `2` 分别生成 `[2]`，导致重复。

解决方式：

1. 先排序，让重复元素相邻；
2. 在同一层循环里，如果当前数字和前一个数字相同，并且前一个数字已经被同层跳过，就跳过当前数字。

```python
if i > start and nums[i] == nums[i - 1]:
    continue
```

---

## 代码实现

```python
from typing import List


class Solution:
    def subsetsWithDup(self, nums: List[int]) -> List[List[int]]:
        nums.sort()
        res = []
        path = []

        def backtrack(start: int) -> None:
            res.append(path[:])

            for i in range(start, len(nums)):
                if i > start and nums[i] == nums[i - 1]:
                    continue

                path.append(nums[i])
                backtrack(i + 1)
                path.pop()

        backtrack(0)
        return res
```

---

## 易错点与复盘

- **每个节点都要记录答案。** 子集不是等到叶子节点才记录。
- **去重只发生在同层。** `i > start` 是关键，不是简单的 `i > 0`。
- **先排序再去重。** 没排序时，相同数字不一定相邻。
- **递归传 `i + 1`。** 子集中每个下标只能使用一次。

## 示例

```text
nums = [1,2,2]

排序后仍是：
[1,2,2]

所有不重复子集：
[]
[1]
[1,2]
[1,2,2]
[2]
[2,2]
```

如果没有同层去重，两个 `2` 会分别生成 `[2]`，最终出现重复子集。
