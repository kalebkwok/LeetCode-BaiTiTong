# 329. 矩阵中的最长递增路径 (Hard)

> **专题归类：** [[02-Wiki/专题总结/10-动态规划]]
> **LeetCode 链接：** https://leetcode.cn/problems/longest-increasing-path-in-a-matrix/

## 题目目标

在矩阵中上下左右移动，求严格递增路径最大长度。

## 重点

- 严格递增边不会成环，可用记忆化 DFS。
- `dfs(r,c)` 表示从该格开始的最长路径。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(mn) |
| 空间复杂度 | O(mn) |

## 代码实现

```python
from functools import cache
from typing import List


class Solution:
    def longestIncreasingPath(self, matrix: List[List[int]]) -> int:
        rows, cols = len(matrix), len(matrix[0])

        @cache
        def dfs(r: int, c: int) -> int:
            best = 1
            for dr, dc in ((1, 0), (-1, 0), (0, 1), (0, -1)):
                nr, nc = r + dr, c + dc
                if 0 <= nr < rows and 0 <= nc < cols and matrix[nr][nc] > matrix[r][c]:
                    best = max(best, 1 + dfs(nr, nc))
            return best

        return max(dfs(r, c) for r in range(rows) for c in range(cols))
```

## 易错点与复盘

- 递增限制使图是 DAG；不要用全局 visited 阻断不同路径复用节点。
- 缓存的是“从此处开始”的结果。

## 示例

```text
[[9,9,4],[6,6,8],[2,1,1]] 的最长路径为 1→2→6→9，长度 4。
```
