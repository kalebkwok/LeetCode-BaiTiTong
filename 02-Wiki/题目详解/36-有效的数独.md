# 36. 有效的数独 (Medium)

> **专题归类：** [[02-Wiki/专题总结/03-数组与矩阵]]
> **LeetCode 链接：** https://leetcode.cn/problems/valid-sudoku/

## 题目目标

判断 9×9 数独当前已填数字是否满足每行、每列和每个 3×3 宫内不重复。

## 重点

- 三类约束分别用集合记录：行、列、宫。
- 宫编号是 `(row // 3, col // 3)`。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(1) |
| 空间复杂度 | O(1) |

棋盘尺寸固定；按一般 `n×n` 棋盘则为 O(n²)。

## 解题思路

扫描每个非 `'.'` 的数字；若它已经出现在对应行、列或宫的集合中，立即返回 False，否则三个集合都加入该数字。

## 代码实现

```python
from collections import defaultdict
from typing import List


class Solution:
    def isValidSudoku(self, board: List[List[str]]) -> bool:
        rows, cols, boxes = defaultdict(set), defaultdict(set), defaultdict(set)
        for r in range(9):
            for c in range(9):
                value = board[r][c]
                if value == ".":
                    continue
                box = (r // 3, c // 3)
                if value in rows[r] or value in cols[c] or value in boxes[box]:
                    return False
                rows[r].add(value)
                cols[c].add(value)
                boxes[box].add(value)
        return True
```

## 易错点与复盘

- 这是验证题，不需要求解数独。
- 宫坐标不是 `r * 3 + c`，而是两个整除后的坐标。

## 示例

```text
同一行出现两个 "5" 时，第二个 5 会命中 rows[r]，返回 False。
```
