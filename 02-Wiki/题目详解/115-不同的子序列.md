# 115. 不同的子序列 (Hard)

> **专题归类：** [[02-Wiki/专题总结/10-动态规划]]
> **LeetCode 链接：** https://leetcode.cn/problems/distinct-subsequences/

## 题目目标

统计 `s` 中有多少个不同子序列等于 `t`。

## 重点

- `dp[i][j]` 是 `s[:i]` 形成 `t[:j]` 的方案数。
- 字符相等时可选用或跳过；不等时只能跳过 `s[i-1]`。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(mn) |
| 空间复杂度 | O(mn) |

## 代码实现

```python
class Solution:
    def numDistinct(self, s: str, t: str) -> int:
        dp = [[0] * (len(t) + 1) for _ in range(len(s) + 1)]
        for i in range(len(s) + 1):
            dp[i][0] = 1
        for i in range(1, len(s) + 1):
            for j in range(1, len(t) + 1):
                dp[i][j] = dp[i - 1][j]
                if s[i - 1] == t[j - 1]:
                    dp[i][j] += dp[i - 1][j - 1]
        return dp[-1][-1]
```

## 易错点与复盘

- 空目标串只有一种构造方式：什么都不选。
- 相等时两条转移是相加，不是二选一。

## 示例

```text
s="rabbbit", t="rabbit"，可删不同一个 b，共 3 种。
```
