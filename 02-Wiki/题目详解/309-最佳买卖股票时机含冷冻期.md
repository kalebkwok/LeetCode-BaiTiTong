# 309. 最佳买卖股票时机含冷冻期 (Medium)

> **专题归类：** [[02-Wiki/专题总结/10-动态规划]]
> **LeetCode 链接：** https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-with-cooldown/

## 题目目标

卖出股票后必须冷冻一天，求最大利润。

## 重点

- 三个状态：持有 `hold`、卖出当天 `sold`、空仓可买 `rest`。
- `hold` 只能从昨天持有或昨天可买状态转来，不能从昨天卖出状态买入。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n) |
| 空间复杂度 | O(1) |

## 代码实现

```python
from typing import List


class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        hold, sold, rest = float("-inf"), 0, 0
        for price in prices:
            previous_hold, previous_sold, previous_rest = hold, sold, rest
            hold = max(previous_hold, previous_rest - price)
            sold = previous_hold + price
            rest = max(previous_rest, previous_sold)
        return max(sold, rest)
```

## 易错点与复盘

- 更新状态时必须保存前一天的值，不能串用本轮刚更新的变量。
- 最终不可能持有股票来获得已实现最大利润。

## 示例

```text
[1,2,3,0,2]：买 1 卖 2，冷冻，买 0 卖 2，利润 3。
```
