# 235. 二叉搜索树的最近公共祖先 (Medium)

> **专题归类：** [[02-Wiki/专题总结/05-二叉树]]
> **LeetCode 链接：** https://leetcode.cn/problems/lowest-common-ancestor-of-a-binary-search-tree/

## 题目目标

在二叉搜索树中找到节点 `p` 和 `q` 的最近公共祖先。

## 重点

- 利用 BST：左子树值更小，右子树值更大。
- 两节点在当前值同侧就继续向同侧走；分居两侧时当前节点就是答案。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(h) |
| 空间复杂度 | O(1) |

## 解题思路

令 `low, high` 是 `p.val, q.val` 的较小和较大值。若当前值小于 `low` 去右，若大于 `high` 去左；否则当前值夹在两者之间，正是分叉点。

## 代码实现

```python
class Solution:
    def lowestCommonAncestor(self, root: TreeNode, p: TreeNode, q: TreeNode) -> TreeNode:
        low, high = sorted((p.val, q.val))
        while root:
            if root.val < low:
                root = root.right
            elif root.val > high:
                root = root.left
            else:
                return root
```

## 易错点与复盘

- 这是 BST 专用解法，普通二叉树不能按值直接走向一侧。
- `p` 或 `q` 本身也可能是最近公共祖先。
