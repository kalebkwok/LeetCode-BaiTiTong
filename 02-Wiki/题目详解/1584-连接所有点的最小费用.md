# 1584. 连接所有点的最小费用 (Medium)

> **专题归类：** [[02-Wiki/专题总结/07-图论]]
> **LeetCode 链接：** https://leetcode.cn/problems/min-cost-to-connect-all-points/

## 题目目标

用曼哈顿距离连接所有点，求最小总成本。

## 重点

- 完全图的最小生成树可用 Prim。
- 每轮选到当前树距离最小的未访问点，并松弛所有其他点。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n² log n) |
| 空间复杂度 | O(n) |

## 代码实现

```python
import heapq
from typing import List


class Solution:
    def minCostConnectPoints(self, points: List[List[int]]) -> int:
        heap, seen, total = [(0, 0)], set(), 0
        while len(seen) < len(points):
            cost, i = heapq.heappop(heap)
            if i in seen:
                continue
            seen.add(i)
            total += cost
            x1, y1 = points[i]
            for j, (x2, y2) in enumerate(points):
                if j not in seen:
                    heapq.heappush(heap, (abs(x1 - x2) + abs(y1 - y2), j))
        return total
```

## 易错点与复盘

- 这题不需要显式建出 O(n²) 条边，加入节点时动态计算距离即可。
- 堆里允许旧候选，弹出后用 `seen` 跳过。

## 示例

```text
[[0,0],[2,2],[3,10],[5,2],[7,0]] 的最小连接成本为 20。
```
