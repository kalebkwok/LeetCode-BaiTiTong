# 323. 无向图中连通分量的数目 (Medium)

> **专题归类：** [[02-Wiki/专题总结/07-图论]]
> **LeetCode 链接：** https://leetcode.ca/all/323.html

## 题目目标

计算无向图中彼此不连通的连通分量数量。

## 重点

- 每次从未访问节点 DFS/BFS，会完整覆盖一个分量。
- 启动搜索次数就是分量数。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(V+E) |
| 空间复杂度 | O(V+E) |

## 代码实现

```python
from collections import defaultdict
from typing import List


class Solution:
    def countComponents(self, n: int, edges: List[List[int]]) -> int:
        graph = defaultdict(list)
        for a, b in edges:
            graph[a].append(b); graph[b].append(a)
        seen, components = set(), 0
        for start in range(n):
            if start in seen:
                continue
            components += 1
            stack = [start]
            seen.add(start)
            while stack:
                node = stack.pop()
                for neighbor in graph[node]:
                    if neighbor not in seen:
                        seen.add(neighbor); stack.append(neighbor)
        return components
```

## 易错点与复盘

- 孤立节点也单独算一个分量。
- 图可能不连通，不能只从 0 开始一次 DFS。

## 示例

```text
n=5，边 [[0,1],[1,2],[3,4]] 有两个分量。
```
