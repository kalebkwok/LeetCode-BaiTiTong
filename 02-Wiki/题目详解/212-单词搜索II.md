# 212. 单词搜索 II (Hard)

> **专题归类：** [[02-Wiki/专题总结/07-图论]] · [[02-Wiki/专题总结/08-回溯算法]]
> **LeetCode 链接：** https://leetcode.cn/problems/word-search-ii/

## 题目目标

在字符网格中找出给定列表中所有可由相邻格组成的单词。

## 重点

- 把所有单词放入 Trie，DFS 遇到不存在前缀立即剪枝。
- 当前路径用 `#` 原地标记，递归后恢复。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | 最坏 O(mn·4^L) |
| 空间复杂度 | O(总单词长度 + L) |

## 代码实现

```python
class Solution:
    def findWords(self, board: list[list[str]], words: list[str]) -> list[str]:
        root = {}
        for word in words:
            node = root
            for char in word:
                node = node.setdefault(char, {})
            node["$"] = word
        result, rows, cols = [], len(board), len(board[0])

        def dfs(r: int, c: int, node: dict) -> None:
            char = board[r][c]
            if char not in node:
                return
            node = node[char]
            found = node.pop("$", None)
            if found:
                result.append(found)
            board[r][c] = "#"
            for dr, dc in ((1,0), (-1,0), (0,1), (0,-1)):
                nr, nc = r + dr, c + dc
                if 0 <= nr < rows and 0 <= nc < cols and board[nr][nc] != "#":
                    dfs(nr, nc, node)
            board[r][c] = char

        for r in range(rows):
            for c in range(cols):
                dfs(r, c, root)
        return result
```

## 易错点与复盘

- `pop("$")` 可避免同一单词被重复加入结果。
- 必须恢复网格字符，访问标记只属于当前 DFS 路径。

## 示例

```text
board=[[o,a,a,n],[e,t,a,e],[i,h,k,r],[i,f,l,v]]，words=["oath","pea","eat","rain"]，返回 oath、eat。
```
