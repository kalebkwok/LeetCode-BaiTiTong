# 704. 二分查找 (Easy)

> **专题归类：** [[02-Wiki/专题总结/09-二分查找]]
> **LeetCode 链接：** https://leetcode.cn/problems/binary-search/

## 题目目标

在升序数组中查找 `target` 的下标，不存在则返回 `-1`。

## 重点

- 维护闭区间 `[left, right]`，循环条件是 `left <= right`。
- 比较后排除 `mid`，所以更新为 `mid + 1` 或 `mid - 1`。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(log n) |
| 空间复杂度 | O(1) |

## 解题思路

每轮比较中点。中点值太小，答案只可能在右半；太大只可能在左半。

## 代码实现

```python
from typing import List


class Solution:
    def search(self, nums: List[int], target: int) -> int:
        left, right = 0, len(nums) - 1
        while left <= right:
            mid = left + (right - left) // 2
            if nums[mid] == target:
                return mid
            if nums[mid] < target:
                left = mid + 1
            else:
                right = mid - 1
        return -1
```

## 易错点与复盘

- 闭区间版本的结束条件不是 `left < right`。
- 数组必须有序；二分靠有序性排除半边。
