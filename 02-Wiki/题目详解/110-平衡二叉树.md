# 110. 平衡二叉树 (Easy)

> **专题归类：** [[02-Wiki/专题总结/05-二叉树]]
> **LeetCode 链接：** https://leetcode.cn/problems/balanced-binary-tree/

## 题目目标

判断二叉树任意节点的左右子树高度差是否都不超过 1。

## 重点

- 后序遍历：先拿到左右子树高度，再判断当前节点。
- 用 `-1` 作为“不平衡”的信号，能在发现问题后提前结束。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n) |
| 空间复杂度 | O(h) |

## 解题思路

递归函数返回子树高度；一旦任意子树不平衡就返回 `-1` 并向上传播。

## 代码实现

```python
class Solution:
    def isBalanced(self, root: TreeNode | None) -> bool:
        def height(node: TreeNode | None) -> int:
            if not node:
                return 0
            left = height(node.left)
            right = height(node.right)
            if left == -1 or right == -1 or abs(left - right) > 1:
                return -1
            return max(left, right) + 1

        return height(root) != -1
```

## 易错点与复盘

- 不要为每个节点重新计算高度，否则会退化为 O(n²)。
- `h` 是树高，递归栈最坏为 O(n)。
