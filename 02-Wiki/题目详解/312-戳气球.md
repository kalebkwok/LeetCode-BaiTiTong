# 312. 戳气球 (Hard)

> **专题归类：** [[02-Wiki/专题总结/10-动态规划]]
> **LeetCode 链接：** https://leetcode.cn/problems/burst-balloons/

## 题目目标

戳破所有气球，求可获得硬币数的最大值。

## 重点

- 想“区间里最后戳哪个”，而不是先戳哪个。
- `dp[left][right]` 表示开区间 `(left,right)` 内的最大收益。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n³) |
| 空间复杂度 | O(n²) |

## 代码实现

```python
from typing import List


class Solution:
    def maxCoins(self, nums: List[int]) -> int:
        values = [1] + nums + [1]
        n = len(values)
        dp = [[0] * n for _ in range(n)]
        for length in range(2, n):
            for left in range(n - length):
                right = left + length
                for last in range(left + 1, right):
                    dp[left][right] = max(dp[left][right], values[left] * values[last] * values[right] + dp[left][last] + dp[last][right])
        return dp[0][n - 1]
```

## 易错点与复盘

- 两侧补 1 是为了统一边界。
- `last` 戳破时，区间内其他气球都没了，所以相邻的是 `left`、`right`。

## 示例

```text
[3,1,5,8] 的最大硬币数为 167。
```
