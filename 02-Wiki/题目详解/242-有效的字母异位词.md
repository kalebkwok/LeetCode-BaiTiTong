# 242. 有效的字母异位词 (Easy)

> **专题归类：** [[02-Wiki/专题总结/01-哈希表]]
> **LeetCode 链接：** https://leetcode.cn/problems/valid-anagram/

---

## 题目目标

给定字符串 `s` 和 `t`，判断 `t` 是否是 `s` 的字母异位词。

字母异位词要求两串包含的每种字符及其出现次数完全相同，字符顺序可以不同。

---

## 重点

- 判断异位词，本质是比较两组**字符频次**。
- 先判断长度；长度不同一定不是异位词。
- `Counter(s) == Counter(t)` 会逐个比较键和值，语义直接。

---

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n) |
| 空间复杂度 | O(k) |

`k` 是不同字符数；若字符集固定为小写英文字母，也可视作 O(1) 额外空间。

---

## 解题思路

维护一个频次表：扫描 `s` 时给字符加一，扫描 `t` 时给字符减一。

如果两串的频次完全相同，所有字符最终都会抵消为 0。Python 里可以直接用 `Counter` 完成这个比较。

排序后比较 `sorted(s) == sorted(t)` 也正确，但排序需要 O(n log n)；哈希计数更符合本题的线性解法目标。

---

## 代码实现

```python
from collections import Counter


class Solution:
    def isAnagram(self, s: str, t: str) -> bool:
        if len(s) != len(t):
            return False

        return Counter(s) == Counter(t)
```

---

## 易错点与复盘

- **不要只比较字符集合。** `set(s) == set(t)` 会忽略次数，例如 `"aab"` 和 `"abb"`。
- **长度检查不是必须，但能更早返回。** 长度不同无需构造频次表。
- 若题目限定仅有 `a-z`，可以用长度为 26 的数组替代 `Counter`；通用字符串则优先 `Counter`。

## 示例

```text
s = "anagram", t = "nagaram"

Counter(s) = {a: 3, n: 1, g: 1, r: 1, m: 1}
Counter(t) = {n: 1, a: 3, g: 1, r: 1, m: 1}

频次相同，答案：True
```
