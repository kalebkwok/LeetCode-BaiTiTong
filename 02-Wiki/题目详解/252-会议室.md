# 252. 会议室 (Easy)

> **专题归类：** [[02-Wiki/专题总结/03-数组与矩阵]]
> **LeetCode 链接：** https://leetcode.ca/all/252.html

## 题目目标

判断一个人能否参加全部会议区间。

## 重点

- 按开始时间排序后，只需比较相邻会议。
- 后一场开始时间小于前一场结束时间即冲突。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n log n) |
| 空间复杂度 | O(1) 或 O(n) |

## 代码实现

```python
from typing import List


class Solution:
    def canAttendMeetings(self, intervals: List[List[int]]) -> bool:
        intervals.sort(key=lambda interval: interval[0])
        for i in range(1, len(intervals)):
            if intervals[i][0] < intervals[i - 1][1]:
                return False
        return True
```

## 易错点与复盘

- `[0,10]` 与 `[10,20]` 可以连续参加，不是重叠。
- 不排序不能只看原数组相邻会议。

## 示例

```text
[[0,30],[5,10],[15,20]] 中前两场重叠，返回 False。
```
