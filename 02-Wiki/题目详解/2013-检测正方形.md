# 2013. 检测正方形 (Medium)

> **专题归类：** [[02-Wiki/专题总结/03-数组与矩阵]]
> **LeetCode 链接：** https://leetcode.cn/problems/detect-squares/

## 题目目标

动态加入二维点，并统计以给定点为一个顶点的轴对齐正方形数量。

## 重点

- 用计数器保留重复点；重复点会产生多种组合。
- 枚举与查询点同一列的对角点，边长由纵坐标差确定。

## 时间/空间复杂度

| 操作 | 复杂度 |
|---|---|
| `add` | O(1) |
| `count` | O(n) |
| 空间 | O(n) |

## 代码实现

```python
from collections import Counter
from typing import List


class DetectSquares:
    def __init__(self):
        self.points = Counter()

    def add(self, point: List[int]) -> None:
        self.points[tuple(point)] += 1

    def count(self, point: List[int]) -> int:
        x, y = point
        total = 0
        for (px, py), count in self.points.items():
            if px != x or py == y:
                continue
            side = py - y
            total += count * self.points[(x + side, y)] * self.points[(x + side, py)]
            total += count * self.points[(x - side, y)] * self.points[(x - side, py)]
        return total
```

## 易错点与复盘

- 只枚举同一列的点；相同行不能构成有面积的正方形。
- `Counter` 的乘法体现重复点的组合数，不能用普通集合替代。

## 示例

```text
已有 (3,10)、(11,2)、(3,2) 时，查询 (11,10) 可构成一个正方形。
```
