# 332. 重新安排行程 (Hard)

> **专题归类：** [[02-Wiki/专题总结/07-图论]] · [[02-Wiki/专题总结/08-回溯算法]]
> **LeetCode 链接：** https://leetcode.cn/problems/reconstruct-itinerary/

## 题目目标

从 JFK 出发使用所有机票恰好一次，返回字典序最小的行程。

## 重点

- 这是有向图欧拉路径，使用 Hierholzer 后序加入节点。
- 每个出发机场用最小堆保证字典序。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(E log E) |
| 空间复杂度 | O(E) |

## 代码实现

```python
import heapq
from collections import defaultdict
from typing import List


class Solution:
    def findItinerary(self, tickets: List[List[str]]) -> List[str]:
        graph = defaultdict(list)
        for source, destination in tickets:
            heapq.heappush(graph[source], destination)
        route = []
        def visit(airport):
            while graph[airport]:
                visit(heapq.heappop(graph[airport]))
            route.append(airport)
        visit("JFK")
        return route[::-1]
```

## 易错点与复盘

- 不能普通前序 DFS；走到尽头后才加入答案才能保证用完所有边。
- 末尾必须反转。

## 示例

```text
JFK→MUC→LHR→SFO→SJC 是给定机票的一条完整行程。
```
