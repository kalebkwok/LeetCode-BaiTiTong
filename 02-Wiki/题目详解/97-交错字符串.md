# 97. 交错字符串 (Medium)

> **专题归类：** [[02-Wiki/专题总结/10-动态规划]]
> **LeetCode 链接：** https://leetcode.cn/problems/interleaving-string/

## 题目目标

判断 `s3` 是否能由 `s1`、`s2` 保持各自相对顺序地交错组成。

## 重点

- `dp[i][j]`：`s1[:i]`、`s2[:j]` 能否组成 `s3[:i+j]`。
- 最后一个字符要么来自 `s1[i-1]`，要么来自 `s2[j-1]`。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(mn) |
| 空间复杂度 | O(mn) |

## 代码实现

```python
class Solution:
    def isInterleave(self, s1: str, s2: str, s3: str) -> bool:
        if len(s1) + len(s2) != len(s3):
            return False
        dp = [[False] * (len(s2) + 1) for _ in range(len(s1) + 1)]
        dp[0][0] = True
        for i in range(len(s1) + 1):
            for j in range(len(s2) + 1):
                if i and s1[i - 1] == s3[i + j - 1]:
                    dp[i][j] |= dp[i - 1][j]
                if j and s2[j - 1] == s3[i + j - 1]:
                    dp[i][j] |= dp[i][j - 1]
        return dp[-1][-1]
```

## 易错点与复盘

- 先检查长度；长度不等无需 DP。
- `s3` 下标是 `i + j - 1`。

## 示例

```text
s1="aab", s2="axy", s3="aaxaby"，返回 True。
```
