# 191. 位 1 的个数 (Easy)

> **专题归类：** [[02-Wiki/专题总结/12-技巧专题]]
> **LeetCode 链接：** https://leetcode.cn/problems/number-of-1-bits/

## 题目目标

返回无符号整数 `n` 的二进制表示中 `1` 的个数。

## 重点

- `n & (n - 1)` 会消去 `n` 最低位的一个 `1`。
- 消去多少次，就有多少个 `1`。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(k) |
| 空间复杂度 | O(1) |

`k` 是二进制中 `1` 的数量。

## 解题思路

每次执行 `n &= n - 1`，最低位的 `1` 都会变成 `0`。计数器累加到 `n` 变为 0 为止。

## 代码实现

```python
class Solution:
    def hammingWeight(self, n: int) -> int:
        count = 0
        while n:
            n &= n - 1
            count += 1
        return count
```

## 易错点与复盘

- 这是“删最低位 1”，不是右移一位。
- 若只需要结果，Python 的 `n.bit_count()` 也可用；面试中要会解释位运算版本。
