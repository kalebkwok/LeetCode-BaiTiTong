# 125. 验证回文串 (Easy)

> **专题归类：** [[02-Wiki/专题总结/02-双指针与滑动窗口]]
> **LeetCode 链接：** https://leetcode.cn/problems/valid-palindrome/

---

## 题目目标

给定字符串 `s`，忽略非字母数字字符并忽略大小写后，判断它是否是回文串。

---

## 重点

- 用左右指针从两端向中间收缩。
- `isalnum()` 决定一个字符是否参与比较；标点和空格只负责被跳过。
- 每次真正比较前都要统一大小写。

---

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n) |
| 空间复杂度 | O(1) |

双指针原地扫描，不需要构造过滤后的新字符串。

---

## 解题思路

令 `left` 指向开头、`right` 指向结尾。循环中先把两端的非字母数字字符跳过去；当两端都停在有效字符上时，再比较它们的小写形式。

- 不同：立刻不是回文；
- 相同：两个指针各向中间移动一步；
- 指针相遇或交错：所有有效字符都已匹配。

---

## 代码实现

```python
class Solution:
    def isPalindrome(self, s: str) -> bool:
        left, right = 0, len(s) - 1

        while left < right:
            while left < right and not s[left].isalnum():
                left += 1
            while left < right and not s[right].isalnum():
                right -= 1

            if s[left].lower() != s[right].lower():
                return False

            left += 1
            right -= 1

        return True
```

---

## 易错点与复盘

- **先跳过无效字符，再比较。** 否则空格、逗号会导致误判。
- 内层跳过循环也要写 `left < right`，避免两指针越过后继续访问。
- 不要只在一端调用 `lower()`；比较的两端都必须忽略大小写。

```text
s = "A man, a plan, a canal: Panama"

a m a n a p l a n a c a n a l p a n a m a

左右始终相等，答案：True
```
