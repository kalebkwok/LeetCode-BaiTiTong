# 190. 颠倒二进制位 (Easy)

> **专题归类：** [[02-Wiki/专题总结/12-技巧专题]]
> **LeetCode 链接：** https://leetcode.cn/problems/reverse-bits/

## 题目目标

把一个 32 位无符号整数的二进制位顺序完全颠倒。

## 重点

- 每轮取出 `n` 的最低位，接到答案的最低端。
- `result = (result << 1) | (n & 1)` 完成“左移腾位 + 填当前位”。
- 必须固定循环 32 次，前导 0 也属于结果位。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(1) |
| 空间复杂度 | O(1) |

## 解题思路

从右到左读取原数，把读出的位从左到右写入结果。每读一位，原数右移一位，答案左移一位。

## 代码实现

```python
class Solution:
    def reverseBits(self, n: int) -> int:
        result = 0
        for _ in range(32):
            result = (result << 1) | (n & 1)
            n >>= 1
        return result
```

## 易错点与复盘

- 不能写成 `while n`，那会丢失前导 0 的位置。
- `n & 1` 是当前最低位；`n >>= 1` 才是读取后丢掉它。
