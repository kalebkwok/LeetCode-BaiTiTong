# 7. 整数反转 (Medium)

> **专题归类：** [[02-Wiki/专题总结/12-技巧专题]]
> **LeetCode 链接：** https://leetcode.cn/problems/reverse-integer/

## 题目目标

反转 32 位有符号整数 `x` 的十进制数字；若结果越界，返回 0。

## 重点

- 每轮取末位，执行 `result = result * 10 + digit`。
- Python 没有整数溢出，但题目要求手动检查 32 位范围。
- 先处理符号，避免 Python 负数整除的取整语义干扰取位。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(log |x|) |
| 空间复杂度 | O(1) |

## 解题思路

保存符号后处理绝对值。不断用 `divmod(x, 10)` 取出个位并拼接到答案末尾；最终乘回符号，并检查是否落在 `[−2^31, 2^31−1]`。

## 代码实现

```python
class Solution:
    def reverse(self, x: int) -> int:
        sign = -1 if x < 0 else 1
        x = abs(x)
        result = 0

        while x:
            x, digit = divmod(x, 10)
            result = result * 10 + digit

        result *= sign
        return result if -(2 ** 31) <= result <= 2 ** 31 - 1 else 0
```

## 易错点与复盘

- 反转 `120` 得到 `21`，前导 0 不需要保留。
- 不要只检查正数上界；负数下界是 `-2^31`。
