# 746. 使用最小花费爬楼梯 (Easy)

> **专题归类：** [[02-Wiki/专题总结/10-动态规划]]
> **LeetCode 链接：** https://leetcode.cn/problems/min-cost-climbing-stairs/

## 题目目标

每次爬 1 或 2 级，踏上台阶 `i` 需付 `cost[i]`，求到达楼顶的最小花费。

## 重点

- 到达当前台阶的最小成本只依赖前两个状态。
- 楼顶可从最后一级或倒数第二级跨到，不再付费用。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n) |
| 空间复杂度 | O(1) |

## 代码实现

```python
from typing import List


class Solution:
    def minCostClimbingStairs(self, cost: List[int]) -> int:
        first = second = 0
        for value in cost:
            first, second = second, min(first, second) + value
        return min(first, second)
```

## 易错点与复盘

- `first`、`second` 表示到达前两级的成本，不是“从某级出发”的成本。
- 最终返回两个状态的最小值。

## 示例

```text
cost=[10,15,20]，可从第 1 级跨到顶端，最小成本为 15。
```
