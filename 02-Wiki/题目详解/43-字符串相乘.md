# 43. 字符串相乘 (Medium)

> **专题归类：** [[02-Wiki/专题总结/03-数组与矩阵]]
> **LeetCode 链接：** https://leetcode.cn/problems/multiply-strings/

## 题目目标

不使用大整数库，计算两个非负整数字符串的乘积。

## 重点

- 竖式乘法中 `num1[i] * num2[j]` 影响结果数组的 `i+j+1` 位。
- 长度为 `m+n` 的数组足够保存结果。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(mn) |
| 空间复杂度 | O(m+n) |

## 代码实现

```python
class Solution:
    def multiply(self, num1: str, num2: str) -> str:
        if num1 == "0" or num2 == "0":
            return "0"
        result = [0] * (len(num1) + len(num2))
        for i in range(len(num1) - 1, -1, -1):
            for j in range(len(num2) - 1, -1, -1):
                product = (ord(num1[i]) - ord("0")) * (ord(num2[j]) - ord("0"))
                pos = i + j + 1
                total = result[pos] + product
                result[pos] = total % 10
                result[pos - 1] += total // 10
        return "".join(map(str, result)).lstrip("0")
```

## 易错点与复盘

- 乘积要先加上该位置已有的进位值。
- 去掉前导零后，零乘法必须提前返回，否则会得到空串。

## 示例

```text
"123" × "45" 按每对数字累加到对应的十进制位，结果为 "5535"。
```
