# 213. 打家劫舍 II (Medium)

> **专题归类：** [[02-Wiki/专题总结/10-动态规划]]
> **LeetCode 链接：** https://leetcode.cn/problems/house-robber-ii/

## 题目目标

房屋首尾相邻，不能偷相邻两家，求最大金额。

## 重点

- 环形约束拆成两条线：不偷首家或不偷末家。
- 线性子问题复用 198. 打家劫舍的转移。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n) |
| 空间复杂度 | O(1) |

## 代码实现

```python
from typing import List


class Solution:
    def rob(self, nums: List[int]) -> int:
        if len(nums) == 1:
            return nums[0]

        def rob_line(values: List[int]) -> int:
            previous = current = 0
            for value in values:
                previous, current = current, max(current, previous + value)
            return current

        return max(rob_line(nums[1:]), rob_line(nums[:-1]))
```

## 易错点与复盘

- 不能同时选首尾，所以不应直接对整段跑线性 DP。
- 只有一间房要单独处理。

## 示例

```text
[2,3,2] 不能同时拿两端 2，最优是 3。
```
