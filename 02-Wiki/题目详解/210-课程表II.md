# 210. 课程表 II (Medium)

> **专题归类：** [[02-Wiki/专题总结/07-图论]]
> **LeetCode 链接：** https://leetcode.cn/problems/course-schedule-ii/

## 题目目标

给出课程先修关系，返回可完成全部课程的一种顺序；有环则返回空数组。

## 重点

- Kahn 算法：入度为 0 的课程可先学。
- 最终输出数少于课程总数说明图有环。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(V+E) |
| 空间复杂度 | O(V+E) |

## 代码实现

```python
from collections import deque
from typing import List


class Solution:
    def findOrder(self, numCourses: int, prerequisites: List[List[int]]) -> List[int]:
        graph = [[] for _ in range(numCourses)]
        degree = [0] * numCourses
        for course, prerequisite in prerequisites:
            graph[prerequisite].append(course)
            degree[course] += 1
        queue = deque(i for i in range(numCourses) if degree[i] == 0)
        order = []
        while queue:
            course = queue.popleft()
            order.append(course)
            for next_course in graph[course]:
                degree[next_course] -= 1
                if degree[next_course] == 0:
                    queue.append(next_course)
        return order if len(order) == numCourses else []
```

## 易错点与复盘

- 边方向是“先修课 → 课程”。
- 207 只问能否完成；210 需要保留出队顺序。

## 示例

```text
2 门课且 [1,0] 表示先学 0，答案为 [0,1]。
```
