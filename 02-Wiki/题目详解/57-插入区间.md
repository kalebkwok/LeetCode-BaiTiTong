# 57. 插入区间 (Medium)

> **专题归类：** [[02-Wiki/专题总结/03-数组与矩阵]]
> **LeetCode 链接：** https://leetcode.cn/problems/insert-interval/

## 题目目标

向按起点排序且互不重叠的区间列表插入 `newInterval`，合并所有重叠区间。

## 重点

- 区间分为：新段左侧、不重叠合并段、新段右侧。
- 相接的端点也算重叠，条件用 `interval[0] <= end`。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n) |
| 空间复杂度 | O(n) |

## 解题思路

先收集结束位置严格小于新段起点的区间；随后不断扩张新段直到下一个区间不重叠；最后接上剩余区间。

## 代码实现

```python
from typing import List


class Solution:
    def insert(self, intervals: List[List[int]], newInterval: List[int]) -> List[List[int]]:
        result, i = [], 0
        start, end = newInterval
        while i < len(intervals) and intervals[i][1] < start:
            result.append(intervals[i])
            i += 1
        while i < len(intervals) and intervals[i][0] <= end:
            start = min(start, intervals[i][0])
            end = max(end, intervals[i][1])
            i += 1
        result.append([start, end])
        return result + intervals[i:]
```

## 易错点与复盘

- 左侧判定用 `< start`，不是 `<= start`；端点相接要进入合并段。
- 已有区间有序才可一次线性扫描。

## 示例

```text
[[1,3],[6,9]] 插入 [2,5]，合并后为 [[1,5],[6,9]]。
```
