# 695. 岛屿的最大面积 (Medium)

> **专题归类：** [[02-Wiki/专题总结/07-图论]]
> **LeetCode 链接：** https://leetcode.cn/problems/max-area-of-island/

## 题目目标

求网格中由 1 组成的最大连通岛屿面积。

## 重点

- 每次 DFS/BFS 计算一个连通块大小。
- 访问过的陆地改为 0，避免重复计数。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(mn) |
| 空间复杂度 | O(mn) |

## 代码实现

```python
from typing import List


class Solution:
    def maxAreaOfIsland(self, grid: List[List[int]]) -> int:
        rows, cols = len(grid), len(grid[0])
        def dfs(r: int, c: int) -> int:
            if not (0 <= r < rows and 0 <= c < cols) or grid[r][c] == 0:
                return 0
            grid[r][c] = 0
            return 1 + sum(dfs(r + dr, c + dc) for dr, dc in ((1,0),(-1,0),(0,1),(0,-1)))
        return max((dfs(r, c) for r in range(rows) for c in range(cols)), default=0)
```

## 易错点与复盘

- 边界检查要在访问 `grid[r][c]` 前。
- 空网格可返回 0；题目通常保证非空。

## 示例

```text
一块相连的 1 有 6 格，答案为 6。
```
