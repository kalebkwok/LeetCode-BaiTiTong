# 40. 组合总和 II (Medium)

> **专题归类：** [[02-Wiki/专题总结/08-回溯算法]]
> **LeetCode 链接：** https://leetcode.cn/problems/combination-sum-ii/

---

## 题目目标

给定一个可能包含重复数字的数组 `candidates` 和目标值 `target`，找出所有和为 `target` 的组合。

每个数字在每个组合中只能使用一次，结果中不能包含重复组合。

---

## 重点

- 这是 [[02-Wiki/题目详解/39-组合总和|39-组合总和]] 的进阶版：**不能重复使用同一个元素**。
- `candidates` 里可能有重复数字，所以必须处理重复组合。
- 排序后用 `start` 控制向后选择，用同层去重跳过重复分支。
- 去重条件是：`if i > start and candidates[i] == candidates[i - 1]: continue`。

---

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n · 2^n) |
| 空间复杂度 | O(n) |

不计输出结果空间。最坏情况下需要枚举大量子集，每次拷贝路径最多 O(n)。

---

## 解题思路

这题有两个核心限制：

1. 每个下标只能用一次；
2. 相同数值不能导致重复组合。

第一个限制靠 `start`：

```python
backtrack(i + 1, ...)
```

传 `i + 1` 表示下一层不能再使用当前下标。

第二个限制靠“同层去重”。

排序后，相同数字会挨在一起。如果在同一层中，前一个相同数字已经作为这一层的选择起点，那么当前相同数字再作为起点会生成重复组合。

```python
if i > start and candidates[i] == candidates[i - 1]:
    continue
```

注意这里是 `i > start`，表示“同一层”的重复才跳过；不同层可以使用相同数值，因为它们来自不同下标。

---

## 代码实现

```python
from typing import List


class Solution:
    def combinationSum2(self, candidates: List[int], target: int) -> List[List[int]]:
        candidates.sort()
        res = []
        path = []

        def backtrack(start: int, remaining: int) -> None:
            if remaining == 0:
                res.append(path[:])
                return

            for i in range(start, len(candidates)):
                if i > start and candidates[i] == candidates[i - 1]:
                    continue

                if candidates[i] > remaining:
                    break

                path.append(candidates[i])
                backtrack(i + 1, remaining - candidates[i])
                path.pop()

        backtrack(0, target)
        return res
```

---

## 易错点与复盘

- **递归传 `i + 1`，不是 `i`。** 本题每个元素只能用一次；39 题可以重复用，所以传 `i`。
- **去重条件必须是 `i > start`。** 只跳过同层重复，不跳过不同层重复。
- **排序是去重和剪枝的前提。** 没排序就不能用相邻元素判断重复，也不能安全 `break`。
- **答案要拷贝路径。** 用 `res.append(path[:])`，不要直接保存 `path` 引用。

## 示例

```text
candidates = [10,1,2,7,6,1,5], target = 8

排序后：
[1,1,2,5,6,7,10]

合法组合：
[1,1,6]
[1,2,5]
[1,7]
[2,6]
```

注意 `[1,7]` 只能出现一次。两个 `1` 虽然来自不同下标，但在同一层不能重复开启完全相同的搜索分支。
