# 269. 火星词典 (Hard)

> **专题归类：** [[02-Wiki/专题总结/07-图论]]
> **LeetCode 链接：** https://leetcode.ca/all/269.html

## 题目目标

从已按未知字母顺序排序的单词列表，推导一种合法字符顺序。

## 重点

- 相邻单词第一个不同字符给出唯一边 `a → b`。
- 前长后短且前缀相同时非法；最后用拓扑排序检测环。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(总字符数) |
| 空间复杂度 | O(字母数 + 边数) |

## 代码实现

```python
from collections import defaultdict, deque
from typing import List


class Solution:
    def alienOrder(self, words: List[str]) -> str:
        graph = {char: set() for word in words for char in word}
        degree = {char: 0 for char in graph}
        for first, second in zip(words, words[1:]):
            if len(first) > len(second) and first.startswith(second):
                return ""
            for a, b in zip(first, second):
                if a != b:
                    if b not in graph[a]:
                        graph[a].add(b); degree[b] += 1
                    break
        queue = deque(char for char in degree if degree[char] == 0)
        order = []
        while queue:
            char = queue.popleft(); order.append(char)
            for next_char in graph[char]:
                degree[next_char] -= 1
                if degree[next_char] == 0:
                    queue.append(next_char)
        return "".join(order) if len(order) == len(graph) else ""
```

## 易错点与复盘

- 只比较每对单词的第一个不同字符。
- 前缀非法需要在逐字符比较前判断。

## 示例

```text
["wrt","wrf","er","ett","rftt"] 可推出 "wertf"。
```
