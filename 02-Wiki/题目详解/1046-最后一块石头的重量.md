# 1046. 最后一块石头的重量 (Easy)

> **专题归类：** [[02-Wiki/专题总结/06-栈与堆]]
> **LeetCode 链接：** https://leetcode.cn/problems/last-stone-weight/

---

## 题目目标

每次取出最重的两块石头碰撞：

- 如果重量相等，两块都消失；
- 如果重量不同，剩下一块重量为二者差值的新石头。

返回最后剩下石头的重量；如果没有石头，返回 0。

---

## 重点

- 每轮都要快速取出最大值。
- Python 只有小根堆，所以用负数模拟大根堆。
- 每次弹出两个最大值，必要时把差值再放回堆。

---

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n log n) |
| 空间复杂度 | O(n) |

---

## 解题思路

这题是典型的“动态取最大值”问题，用堆。

Python 的 `heapq` 是小根堆，要取最大值，可以把每个重量取负数放进去：

```text
原重量：  8, 7, 4
堆里存： -8, -7, -4
```

每次 `heappop` 出来的最小负数，对应原来的最大重量。

---

## 代码实现

```python
from typing import List
import heapq


class Solution:
    def lastStoneWeight(self, stones: List[int]) -> int:
        heap = [-stone for stone in stones]
        heapq.heapify(heap)

        while len(heap) > 1:
            y = -heapq.heappop(heap)  # 最大
            x = -heapq.heappop(heap)  # 第二大

            if y != x:
                heapq.heappush(heap, -(y - x))

        return -heap[0] if heap else 0
```

---

## 易错点与复盘

- **负号要成对出现。** 入堆取负，出堆再取负。
- **两个石头一样重时不要放回 0。** 它们会一起消失。
- **最后堆可能为空。** 所以返回时要判断 `heap` 是否为空。

stones = [2,7,4,1,8,1]

答案：1
