# 371. 两整数之和 (Medium)

> **专题归类：** [[02-Wiki/专题总结/12-技巧专题]]
> **LeetCode 链接：** https://leetcode.cn/problems/sum-of-two-integers/

## 题目目标

不使用 `+` 和 `-`，计算两个整数 `a` 与 `b` 的和。

## 重点

- `a ^ b` 得到不带进位的和。
- `(a & b) << 1` 得到进位。
- Python 需要用 32 位掩码模拟补码，才能正确处理负数。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(1) |
| 空间复杂度 | O(1) |

## 解题思路

不断把“无进位和”与“进位”相加，直到没有进位。掩码保证中间结果停留在 32 位无符号范围内，最后再转回 Python 的有符号整数。

## 代码实现

```python
class Solution:
    def getSum(self, a: int, b: int) -> int:
        mask = 0xFFFFFFFF
        max_int = 0x7FFFFFFF

        while b:
            a, b = (a ^ b) & mask, ((a & b) << 1) & mask

        return a if a <= max_int else ~(a ^ mask)
```

## 易错点与复盘

- `a & b` 是进位来源，必须左移一位。
- 没有 `mask` 时，Python 负数的无限符号位会使循环无法按 32 位规则结束。

```text
5 + 3：0101 ^ 0011 = 0110，进位为 0010；0110 + 0010 = 1000。
```
