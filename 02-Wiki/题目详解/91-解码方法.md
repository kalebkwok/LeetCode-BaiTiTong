# 91. 解码方法 (Medium)

> **专题归类：** [[02-Wiki/专题总结/10-动态规划]]
> **LeetCode 链接：** https://leetcode.cn/problems/decode-ways/

## 题目目标

数字字符串按 1→A 到 26→Z 解码，求不同解码数量。

## 重点

- `dp[i]` 表示前 `i` 个字符的解码方式数。
- 当前位非 0 可接一位；前两位在 10~26 可接两位。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n) |
| 空间复杂度 | O(1) |

## 代码实现

```python
class Solution:
    def numDecodings(self, s: str) -> int:
        if not s or s[0] == "0":
            return 0
        two_back, one_back = 1, 1
        for i in range(1, len(s)):
            current = 0
            if s[i] != "0":
                current += one_back
            if 10 <= int(s[i - 1:i + 1]) <= 26:
                current += two_back
            two_back, one_back = one_back, current
        return one_back
```

## 易错点与复盘

- 单独的 `0` 无法解码；`10` 和 `20` 只能作为两位数。
- `two_back` 对应 `dp[i-1]`，因为两位转移跨过两个字符。

## 示例

```text
"226" 可解为 BZ、VF、BBF，共 3 种。
```
