# 1851. 包含每个查询的最小区间 (Hard)

> **专题归类：** [[02-Wiki/专题总结/03-数组与矩阵]] · [[02-Wiki/专题总结/06-栈与堆]]
> **LeetCode 链接：** https://leetcode.cn/problems/minimum-interval-to-include-each-query/

## 题目目标

对每个查询值，找出包含它的最短区间长度；没有则返回 -1。

## 重点

- 区间按左端点、查询按值排序。
- 堆存已开始且候选的 `(长度, 右端点)`，先清除右端点过小的区间。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O((n+q) log n) |
| 空间复杂度 | O(n) |

## 代码实现

```python
import heapq
from typing import List


class Solution:
    def minInterval(self, intervals: List[List[int]], queries: List[int]) -> List[int]:
        intervals.sort()
        answer, heap, i = {}, [], 0
        for query in sorted(queries):
            while i < len(intervals) and intervals[i][0] <= query:
                left, right = intervals[i]
                heapq.heappush(heap, (right - left + 1, right))
                i += 1
            while heap and heap[0][1] < query:
                heapq.heappop(heap)
            answer[query] = heap[0][0] if heap else -1
        return [answer[query] for query in queries]
```

## 易错点与复盘

- 长度是 `right - left + 1`。
- 堆顶只保证最短，先删过期区间后才可作为答案。

## 示例

```text
intervals=[[1,4],[2,4],[3,6],[4,4]]，query=4 的最短区间是 [4,4]，长度 1。
```
