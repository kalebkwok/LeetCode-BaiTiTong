# 202. 快乐数 (Easy)

> **专题归类：** [[02-Wiki/专题总结/12-技巧专题]]
> **LeetCode 链接：** https://leetcode.cn/problems/happy-number/

## 题目目标

反复把正整数替换为各位平方和；判断它能否最终到达 `1`。

## 重点

- 状态不是 `n` 本身，而是“各位平方和”后的下一个数。
- 到不了 `1` 就一定会进入循环，用集合判重即可。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(log n) |
| 空间复杂度 | O(log n) |

## 解题思路

`seen` 存已经出现过的状态。若当前数为 `1`，成功；若再次进入 `seen`，说明循环且永远到不了 `1`。

## 代码实现

```python
class Solution:
    def isHappy(self, n: int) -> bool:
        seen = set()

        while n != 1 and n not in seen:
            seen.add(n)
            total = 0
            while n:
                n, digit = divmod(n, 10)
                total += digit * digit
            n = total

        return n == 1
```

## 易错点与复盘

- 每一轮都要把新平方和赋回 `n`。
- 循环检测要检查的是状态值，而不是原始输入。
