# 268. 丢失的数字 (Easy)

> **专题归类：** [[02-Wiki/专题总结/12-技巧专题]]
> **LeetCode 链接：** https://leetcode.cn/problems/missing-number/

## 题目目标

数组 `nums` 包含 `[0, n]` 中的 `n` 个不同数字，找出缺失的那个数。

## 重点

- 异或满足 `a ^ a = 0`、`a ^ 0 = a`。
- 把完整范围和数组元素全部异或，成对数字抵消后只剩缺失数。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n) |
| 空间复杂度 | O(1) |

## 解题思路

初始令 `missing = n`。遍历数组时同时异或下标 `i` 和元素 `num`；范围 `[0, n-1]` 与数组中已有数字都会抵消。

## 代码实现

```python
from typing import List


class Solution:
    def missingNumber(self, nums: List[int]) -> int:
        missing = len(nums)
        for i, num in enumerate(nums):
            missing ^= i ^ num
        return missing
```

## 易错点与复盘

- 初始值要包含最后一个候选数 `n`。
- 加法求和也能做，但异或不会有大整数溢出问题，且空间为 O(1)。

## 示例

```text
nums = [3, 0, 1]
3 ^ 0 ^ 3 ^ 1 ^ 0 ^ 2 ^ 1 = 2，答案为 2。
```
