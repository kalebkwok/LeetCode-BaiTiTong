# 127. 单词接龙 (Hard)

> **专题归类：** [[02-Wiki/专题总结/07-图论]]
> **LeetCode 链接：** https://leetcode.cn/problems/word-ladder/

## 题目目标

每次只改一个字符，求从 `beginWord` 到 `endWord` 的最短转换序列长度。

## 重点

- 所有边权相同，最短步数用 BFS。
- 生成每个位置的 26 个替换字符作为邻居。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(26 × L × N) |
| 空间复杂度 | O(N) |

## 代码实现

```python
from collections import deque
from typing import List


class Solution:
    def ladderLength(self, beginWord: str, endWord: str, wordList: List[str]) -> int:
        words = set(wordList)
        if endWord not in words:
            return 0
        queue = deque([(beginWord, 1)])
        while queue:
            word, steps = queue.popleft()
            if word == endWord:
                return steps
            for i in range(len(word)):
                for char in "abcdefghijklmnopqrstuvwxyz":
                    candidate = word[:i] + char + word[i + 1:]
                    if candidate in words:
                        words.remove(candidate)
                        queue.append((candidate, steps + 1))
        return 0
```

## 易错点与复盘

- 入队时从集合删除，避免同一词被重复入队。
- 返回长度包含起始词，所以初始步数为 1。

## 示例

```text
hit → hot → dot → dog → cog，长度为 5。
```
