# 435. 无重叠区间 (Medium)

> **专题归类：** [[02-Wiki/专题总结/03-数组与矩阵]] · [[02-Wiki/专题总结/11-贪心算法]]
> **LeetCode 链接：** https://leetcode.cn/problems/non-overlapping-intervals/

## 题目目标

移除最少区间，使剩余区间两两不重叠。

## 重点

- 按结束时间升序保留区间，能留下最多不重叠区间。
- 每遇到重叠区间就计一次删除；保留结束更早的那个。

## 时间/空间复杂度

| 项目 | 复杂度 |
|---|---|
| 时间复杂度 | O(n log n) |
| 空间复杂度 | O(1) 或 O(n) |

排序实现决定额外空间。

## 解题思路

选择结束最早的区间可给后面留下最大的可用空间。若当前起点小于上一保留区间的结束位置，则二者重叠，删当前区间即可。

## 代码实现

```python
from typing import List


class Solution:
    def eraseOverlapIntervals(self, intervals: List[List[int]]) -> int:
        intervals.sort(key=lambda interval: interval[1])
        removed, previous_end = 0, float("-inf")
        for start, end in intervals:
            if start < previous_end:
                removed += 1
            else:
                previous_end = end
        return removed
```

## 易错点与复盘

- `start == previous_end` 不重叠；条件是严格小于。
- 不要按起点排序后随意删，正确的贪心依据是最早结束时间。

## 示例

```text
[[1,2],[2,3],[3,4],[1,3]] 只需删 [1,3]，答案 1。
```
