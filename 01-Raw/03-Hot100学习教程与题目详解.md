## 1. 哈希表

### 核心知识点

哈希表是算法面试中最基础也最常用的数据结构。它的核心价值是**将查找时间从 O(n) 降低到 O(1)**。

**四大应用场景：**

| 场景 | 说明 | 典型题目 |
|------|------|----------|
| 查找 | 快速判断元素是否存在 | 两数之和 |
| 去重 | 判断元素是否重复出现 | 存在重复元素 |
| 计数 | 统计元素出现次数 | 字母异位词分组 |
| 分组 | 按某种规则将元素分组 | 最长连续序列 |

**Python 中的哈希表工具：**

```python
# 1. dict（普通字典）
d = {}
d["key"] = "value"
if "key" in d:        # O(1) 查找
    val = d["key"]

# 2. Counter（计数器）
from collections import Counter
c = Counter([1, 2, 2, 3, 3, 3])  # Counter({3: 3, 2: 2, 1: 1})
c.most_common(2)                    # [(3, 3), (2, 2)]

# 3. defaultdict（带默认值的字典）
from collections import defaultdict
dd = defaultdict(list)   # 默认值为空列表
dd["key"].append(1)      # 无需判断key是否存在

dd2 = defaultdict(int)    # 默认值为0
dd2["key"] += 1           # 无需初始化
```

### 模板代码

```python
# 模板1：两数之和类（查找互补元素）
def hash_lookup(nums, target):
    seen = {}  # 值 -> 索引
    for i, num in enumerate(nums):
        complement = target - num
        if complement in seen:
            return [seen[complement], i]
        seen[num] = i
    return []

# 模板2：计数+分组
def hash_group(strs):
    groups = defaultdict(list)
    for s in strs:
        key = tuple(sorted(s))  # 用排序结果作为分组key
        groups[key].append(s)
    return list(groups.values())
```

## 2. 双指针

### 核心知识点

双指针的核心思想是**利用两个指针的相对运动来减少搜索空间**，将 O(n^2) 降低到 O(n)。

**两种经典模式：**

1. **对撞指针（左右指针）**：两个指针从两端向中间移动，适用于有序数组
2. **快慢指针**：两个指针同向移动但速度不同，适用于链表和数组中的位置关系

### 模板代码

```python
# 模板1：对撞指针（有序数组）
def two_pointers_collision(nums, target):
    left, right = 0, len(nums) - 1
    while left < right:
        s = nums[left] + nums[right]
        if s == target:
            return [left, right]
        elif s < target:
            left += 1    # 和太小，左指针右移增大
        else:
            right -= 1   # 和太大，右指针左移减小
    return []

# 模板2：快慢指针（链表中找环/中点）
def fast_slow_pointer(head):
    slow = fast = head
    while fast and fast.next:
        slow = slow.next          # 慢指针走一步
        fast = fast.next.next     # 快指针走两步
    # 如果有环，fast会追上slow
    # 如果无环，fast到达末尾时slow在中点
    return slow

# 模板3：同向双指针（原地修改数组）
def remove_duplicates(nums):
    slow = 0
    for fast in range(len(nums)):
        if nums[fast] != nums[slow]:
            slow += 1
            nums[slow] = nums[fast]
    return slow + 1
```

## 3. 滑动窗口

### 核心知识点

滑动窗口本质上是**双指针的变体**，用于解决连续子数组/子串问题。

**labuladong 三问判断法：**
1. 什么时候扩大窗口？（右指针移动的条件）
2. 什么时候缩小窗口？（左指针移动的条件）
3. 答案在什么时候更新？（扩大时还是缩小时）

### 模板代码

```python
# 模板1：定长滑动窗口（窗口大小固定为k）
def fixed_window(nums, k):
    window_sum = sum(nums[:k])  # 第一个窗口
    max_sum = window_sum
    for i in range(k, len(nums)):
        window_sum += nums[i] - nums[i - k]  # 滑入新元素，滑出旧元素
        max_sum = max(max_sum, window_sum)
    return max_sum

# 模板2：不定长滑动窗口（通用模板）
def sliding_window(s):
    from collections import defaultdict
    window = defaultdict(int)  # 窗口内的数据
    need = defaultdict(int)    # 需要满足的条件

    left = 0
    result = 0  # 或 float('inf')，取决于求最大还是最小

    for right in range(len(s)):
        # 1. 扩大窗口：右指针元素进入窗口
        c = s[right]
        window[c] += 1

        # 2. 判断是否需要缩小窗口
        while "窗口需要缩小的条件":
            # 3. 缩小窗口：左指针元素离开窗口
            d = s[left]
            window[d] -= 1
            left += 1

        # 4. 更新答案（根据题意决定在扩大还是缩小时更新）
        result = max(result, right - left + 1)

    return result

# 模板3：求最长满足条件的子串（具体示例）
def length_of_longest_substring(s):
    char_set = set()
    left = 0
    max_len = 0
    for right in range(len(s)):
        while s[right] in char_set:      # 窗口内有重复字符，需要缩小
            char_set.remove(s[left])
            left += 1
        char_set.add(s[right])           # 加入新字符
        max_len = max(max_len, right - left + 1)
    return max_len
```

## 4. 前缀和

### 核心知识点

前缀和的核心思想是**预处理一个前缀和数组，使得任意区间的和可以在 O(1) 时间内求出**。

- `prefix[i] = nums[0] + nums[1] + ... + nums[i-1]`
- 区间和 `sum(nums[i..j]) = prefix[j+1] - prefix[i]`

**进阶用法**：前缀和 + 哈希表，用于解决"和为K的子数组"类问题。

### 模板代码

```python
# 模板1：前缀和数组
def prefix_sum(nums):
    n = len(nums)
    prefix = [0] * (n + 1)   # prefix[0] = 0, prefix[i] = sum(nums[0..i-1])
    for i in range(n):
        prefix[i + 1] = prefix[i] + nums[i]

    # 查询区间 [i, j] 的和
    def range_sum(i, j):
        return prefix[j + 1] - prefix[i]
    return prefix

# 模板2：前缀和 + 哈希表（求和为k的子数组个数）
def subarray_sum(nums, k):
    prefix_count = defaultdict(int)
    prefix_count[0] = 1     # 前缀和为0出现1次（空前缀）
    current_sum = 0
    count = 0
    for num in nums:
        current_sum += num
        # 如果存在之前的前缀和 = current_sum - k
        # 说明从那个位置到当前位置的子数组和为k
        count += prefix_count[current_sum - k]
        prefix_count[current_sum] += 1
    return count
```

## 5. 单调队列 / 单调栈

### 核心知识点

- **单调队列**：队列内元素单调递增或递减，用于解决**滑动窗口中的最值问题**
- **单调栈**：栈内元素单调递增或递减，用于解决**下一个更大/更小元素问题**

### 模板代码

```python
from collections import deque

# 模板1：单调队列（滑动窗口最大值）
def max_sliding_window(nums, k):
    result = []
    dq = deque()  # 存储索引，队列中的索引对应的值单调递减

    for i in range(len(nums)):
        # 1. 移除窗口外的元素
        while dq and dq[0] < i - k + 1:
            dq.popleft()
        # 2. 维护单调性：移除比当前元素小的队尾元素
        while dq and nums[dq[-1]] <= nums[i]:
            dq.pop()
        # 3. 加入当前元素
        dq.append(i)
        # 4. 记录结果（窗口形成后）
        if i >= k - 1:
            result.append(nums[dq[0]])  # 队首就是最大值
    return result

# 模板2：单调栈（下一个更大元素）
def next_greater_element(nums):
    n = len(nums)
    result = [-1] * n
    stack = []  # 存储索引，栈底到栈顶单调递减

    for i in range(n):
        # 当前元素比栈顶大，说明找到了栈顶元素的"下一个更大元素"
        while stack and nums[stack[-1]] < nums[i]:
            idx = stack.pop()
            result[idx] = nums[i]
        stack.append(i)
    return result

# 模板3：单调栈（环形数组版本，遍历两遍）
def next_greater_element_circular(nums):
    n = len(nums)
    result = [-1] * n
    stack = []
    for i in range(2 * n):
        while stack and nums[stack[-1]] < nums[i % n]:
            idx = stack.pop()
            result[idx] = nums[i % n]
        stack.append(i % n)
    return result
```

## 6. 链表操作

### 核心知识点

**三大核心技巧：**
1. **虚拟头节点（dummy node）**：避免对头节点的特殊处理
2. **快慢指针**：找中点、找环、找第k个节点
3. **链表反转**：迭代法和递归法

### 模板代码

```python
# 链表节点定义
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

# 模板1：虚拟头节点
def use_dummy_node(head):
    dummy = ListNode(0)
    dummy.next = head
    curr = dummy
    while curr.next:
        # 操作 curr.next
        curr = curr.next
    return dummy.next

# 模板2：链表反转（迭代法）
def reverse_list(head):
    prev = None
    curr = head
    while curr:
        nxt = curr.next    # 保存下一个节点
        curr.next = prev   # 反转指针
        prev = curr        # 前进prev
        curr = nxt         # 前进curr
    return prev  # prev是新的头节点

# 模板3：链表反转（递归法）
def reverse_list_recursive(head):
    if not head or not head.next:
        return head
    new_head = reverse_list_recursive(head.next)
    head.next.next = head   # 让下一个节点指向自己
    head.next = None        # 断开原来的指向
    return new_head

# 模板4：快慢指针找链表中点
def find_middle(head):
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
    return slow  # slow就是中点（偶数长度时为中间偏右）

# 模板5：判断链表是否有环
def has_cycle(head):
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
        if slow == fast:
            return True
    return False

# 模板6：找环的入口
def detect_cycle_entry(head):
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
        if slow == fast:
            # 一个指针从头开始，另一个从相遇点开始
            # 两者每次走一步，再次相遇就是入口
            slow = head
            while slow != fast:
                slow = slow.next
                fast = fast.next
            return slow
    return None

# 模板7：合并两个有序链表
def merge_two_lists(l1, l2):
    dummy = ListNode(0)
    curr = dummy
    while l1 and l2:
        if l1.val <= l2.val:
            curr.next = l1
            l1 = l1.next
        else:
            curr.next = l2
            l2 = l2.next
        curr = curr.next
    curr.next = l1 or l2  # 接上剩余部分
    return dummy.next
```

## 7. 二叉树

### 核心知识点

二叉树的题目几乎都可以用**递归**解决。递归的关键是明确：**函数的定义是什么（做什么）、base case是什么、递归关系是什么。**

**二叉树分治法**：通过左右子树的信息来推导当前树的信息（如最大深度、直径、LCA等）。

**BST（二叉搜索树）特性**：左 < 根 < 右，中序遍历是有序的。

### 模板代码

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

# ========== 递归遍历 ==========

# 前序遍历：根 -> 左 -> 右
def preorder(root):
    if not root:
        return []
    return [root.val] + preorder(root.left) + preorder(root.right)

# 中序遍历：左 -> 根 -> 右
def inorder(root):
    if not root:
        return []
    return inorder(root.left) + [root.val] + inorder(root.right)

# 后序遍历：左 -> 右 -> 根
def postorder(root):
    if not root:
        return []
    return postorder(root.left) + postorder(root.right) + [root.val]

# ========== 迭代遍历 ==========

# 前序遍历（迭代）
def preorder_iterative(root):
    if not root:
        return []
    stack, result = [root], []
    while stack:
        node = stack.pop()
        result.append(node.val)
        if node.right:
            stack.append(node.right)   # 先压右，后处理
        if node.left:
            stack.append(node.left)    # 后压左，先处理
    return result

# 中序遍历（迭代）
def inorder_iterative(root):
    stack, result = [], []
    curr = root
    while curr or stack:
        while curr:
            stack.append(curr)
            curr = curr.left        # 一路向左
        curr = stack.pop()
        result.append(curr.val)     # 处理节点
        curr = curr.right           # 转向右子树
    return result

# ========== BFS层序遍历 ==========

from collections import deque

def level_order(root):
    if not root:
        return []
    queue = deque([root])
    result = []
    while queue:
        level_size = len(queue)
        level = []
        for _ in range(level_size):
            node = queue.popleft()
            level.append(node.val)
            if node.left:
                queue.append(node.left)
            if node.right:
                queue.append(node.right)
        result.append(level)
    return result

# ========== 二叉树分治法 ==========

# 最大深度
def max_depth(root):
    if not root:
        return 0
    left_depth = max_depth(root.left)
    right_depth = max_depth(root.right)
    return 1 + max(left_depth, right_depth)

# 翻转二叉树
def invert_tree(root):
    if not root:
        return None
    root.left, root.right = invert_tree(root.right), invert_tree(root.left)
    return root

# 判断对称二叉树
def is_symmetric(root):
    def check(left, right):
        if not left and not right:
            return True
        if not left or not right:
            return False
        return (left.val == right.val and
                check(left.left, right.right) and
                check(left.right, right.left))
    return check(root.left, root.right) if root else True

# 最近公共祖先（LCA）
def lowest_common_ancestor(root, p, q):
    if not root or root == p or root == q:
        return root
    left = lowest_common_ancestor(root.left, p, q)
    right = lowest_common_ancestor(root.right, p, q)
    if left and right:
        return root        # p和q分别在两侧
    return left or right   # p和q在同一侧

# ========== BST操作 ==========

# BST搜索
def search_bst(root, val):
    if not root or root.val == val:
        return root
    if val < root.val:
        return search_bst(root.left, val)
    return search_bst(root.right, val)

# BST验证（中序遍历是否有序）
def is_valid_bst(root):
    prev = [float('-inf')]
    def inorder_check(node):
        if not node:
            return True
        if not inorder_check(node.left):
            return False
        if node.val <= prev[0]:
            return False
        prev[0] = node.val
        return inorder_check(node.right)
    return inorder_check(root)
```

## 8. 图论

### 核心知识点

图论问题主要包含：
- **DFS/BFS遍历**：搜索所有路径、最短路径
- **并查集**：判断连通性、合并集合
- **拓扑排序**：有向无环图（DAG）的排序
- **字典树（Trie）**：前缀匹配、词频统计

### 模板代码

```python
# ========== DFS模板 ==========

def dfs(graph, start):
    visited = set()
    result = []

    def _dfs(node):
        if node in visited:
            return
        visited.add(node)
        result.append(node)
        for neighbor in graph[node]:
            _dfs(neighbor)

    _dfs(start)
    return result

# ========== BFS模板 ==========

from collections import deque, defaultdict

def bfs(graph, start):
    visited = {start}
    queue = deque([start])
    result = []
    while queue:
        node = queue.popleft()
        result.append(node)
        for neighbor in graph[node]:
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append(neighbor)
    return result

# BFS求最短路径（无权图）
def bfs_shortest_path(graph, start, end):
    visited = {start}
    queue = deque([(start, 0)])  # (当前节点, 距离)
    while queue:
        node, dist = queue.popleft()
        if node == end:
            return dist
        for neighbor in graph[node]:
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append((neighbor, dist + 1))
    return -1

# ========== 并查集模板 ==========

class UnionFind:
    def __init__(self, n):
        self.parent = list(range(n))
        self.rank = [0] * n
        self.count = n  # 连通分量个数

    def find(self, x):
        # 路径压缩
        if self.parent[x] != x:
            self.parent[x] = self.find(self.parent[x])
        return self.parent[x]

    def union(self, x, y):
        root_x = self.find(x)
        root_y = self.find(y)
        if root_x == root_y:
            return False
        # 按秩合并
        if self.rank[root_x] < self.rank[root_y]:
            root_x, root_y = root_y, root_x
        self.parent[root_y] = root_x
        if self.rank[root_x] == self.rank[root_y]:
            self.rank[root_x] += 1
        self.count -= 1
        return True

    def connected(self, x, y):
        return self.find(x) == self.find(y)

# ========== 拓扑排序模板（Kahn算法/BFS） ==========

def topological_sort(num_courses, prerequisites):
    # 构建邻接表和入度表
    graph = defaultdict(list)
    in_degree = [0] * num_courses
    for course, pre in prerequisites:
        graph[pre].append(course)
        in_degree[course] += 1

    # 将入度为0的节点入队
    queue = deque([i for i in range(num_courses) if in_degree[i] == 0])
    order = []

    while queue:
        node = queue.popleft()
        order.append(node)
        for neighbor in graph[node]:
            in_degree[neighbor] -= 1
            if in_degree[neighbor] == 0:
                queue.append(neighbor)

    return order if len(order) == num_courses else []  # 空列表表示有环

# ========== 字典树（Trie）模板 ==========

class Trie:
    def __init__(self):
        self.children = {}
        self.is_end = False

    def insert(self, word):
        node = self
        for ch in word:
            if ch not in node.children:
                node.children[ch] = Trie()
            node = node.children[ch]
        node.is_end = True

    def search(self, word):
        node = self
        for ch in word:
            if ch not in node.children:
                return False
            node = node.children[ch]
        return node.is_end

    def starts_with(self, prefix):
        node = self
        for ch in prefix:
            if ch not in node.children:
                return False
            node = node.children[ch]
        return True
```

## 9. 回溯算法

### 核心知识点

回溯算法的本质是**穷举所有可能**，通过**剪枝**来减少不必要的搜索。

**回溯通用框架：**
1. 做选择（加入路径）
2. 递归
3. 撤销选择（回溯）

**三种基本模式**：子集、排列、组合

### 模板代码

```python
# ========== 通用回溯模板 ==========

def backtrack_template(nums):
    result = []
    path = []

    def backtrack(start):
        # 结束条件：满足某个条件时收集结果
        if "满足条件":
            result.append(path[:])   # 注意要拷贝
            return

        for i in range(start, len(nums)):
            # 剪枝（可选）
            if "不满足条件":
                continue
            path.append(nums[i])        # 做选择
            backtrack(i + 1)            # 递归（注意start参数）
            path.pop()                  # 撤销选择（回溯）

    backtrack(0)
    return result

# ========== 子集 ==========

def subsets(nums):
    result = []
    path = []

    def backtrack(start):
        result.append(path[:])          # 每个节点都收集（子集问题）
        for i in range(start, len(nums)):
            path.append(nums[i])
            backtrack(i + 1)
            path.pop()

    backtrack(0)
    return result

# ========== 全排列 ==========

def permute(nums):
    result = []
    path = []
    used = [False] * len(nums)

    def backtrack():
        if len(path) == len(nums):      # 到达叶子节点
            result.append(path[:])
            return
        for i in range(len(nums)):
            if used[i]:
                continue
            # 剪枝：相同元素只在第一次使用时选
            if i > 0 and nums[i] == nums[i-1] and not used[i-1]:
                continue
            used[i] = True
            path.append(nums[i])
            backtrack()
            path.pop()
            used[i] = False

    backtrack()
    return result

# ========== 组合 ==========

def combine(n, k):
    result = []
    path = []

    def backtrack(start):
        if len(path) == k:              # 选够k个就收集
            result.append(path[:])
            return
        # 剪枝：剩余元素不够选k个了
        for i in range(start, n - k + len(path) + 1):
            path.append(i)
            backtrack(i + 1)
            path.pop()

    backtrack(1)
    return result

# ========== 含重复元素的子集（去重） ==========

def subsets_with_dup(nums):
    nums.sort()                         # 先排序！
    result = []
    path = []

    def backtrack(start):
        result.append(path[:])
        for i in range(start, len(nums)):
            # 同一层中跳过重复元素
            if i > start and nums[i] == nums[i-1]:
                continue
            path.append(nums[i])
            backtrack(i + 1)
            path.pop()

    backtrack(0)
    return result
```

## 10. 二分查找

### 核心知识点

二分查找的难点在于**边界条件的处理**。labuladong提出了一套统一模板来解决各种变体。

**核心原则**：
- 搜索区间用 `[left, right]`（闭区间）
- 循环条件用 `left <= right`
- 根据题目要求选择返回 `left`、`right` 或 `mid`

### 模板代码

```python
# 模板1：标准二分查找（找任意一个目标值）
def binary_search(nums, target):
    left, right = 0, len(nums) - 1
    while left <= right:
        mid = left + (right - left) // 2   # 防止溢出
        if nums[mid] == target:
            return mid
        elif nums[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1

# 模板2：搜索左边界（第一个等于target的位置）
def binary_search_left(nums, target):
    left, right = 0, len(nums) - 1
    while left <= right:
        mid = left + (right - left) // 2
        if nums[mid] == target:
            right = mid - 1              # 收缩右边界，锁定左侧
        elif nums[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    # left是第一个>=target的位置
    if left < len(nums) and nums[left] == target:
        return left
    return -1

# 模板3：搜索右边界（最后一个等于target的位置）
def binary_search_right(nums, target):
    left, right = 0, len(nums) - 1
    while left <= right:
        mid = left + (right - left) // 2
        if nums[mid] == target:
            left = mid + 1               # 收缩左边界，锁定右侧
        elif nums[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    # right是最后一个<=target的位置
    if right >= 0 and nums[right] == target:
        return right
    return -1

# 模板4：寻找第一个满足条件的元素（通用）
def binary_search_first_true(n, condition):
    """
    在 [0, n] 中找到第一个使得 condition(x) 为 True 的 x
    假设 condition 具有单调性：False...False True...True
    """
    left, right = 0, n
    while left < right:
        mid = left + (right - left) // 2
        if condition(mid):
            right = mid          # mid可能是答案，继续往左找
        else:
            left = mid + 1       # mid不是答案，往右找
    return left
```

```python
def is_valid(s):
    stack = []
    mapping = {')': '(', ']': '[', '}': '{'}
    for char in s:
        if char in mapping:
            # 遇到右括号，检查栈顶
            top = stack.pop() if stack else '#'
            if mapping[char] != top:
                return False
        else:
            stack.append(char)

# 模板2：字符串解码
def decode_string(s):
    stack = []
    current_num = 0
    current_str = ''

    for char in s:
        if char.isdigit():
            current_num = current_num * 10 + int(char)
        elif char == '[':
            stack.append((current_str, current_num))  # 保存当前状态
            current_str = ''
            current_num = 0
        elif char == ']':
            prev_str, num = stack.pop()
            current_str = prev_str + current_str * num
        else:
            current_str += char
    return current_str

# 模板3：最小栈（辅助栈）
class MinStack:
    def __init__(self):
        self.stack = []
        self.min_stack = []   # 辅助栈，栈顶始终是当前最小值

    def push(self, val):
        self.stack.append(val)
        if not self.min_stack or val <= self.min_stack[-1]:
            self.min_stack.append(val)

    def pop(self):
        if self.stack.pop() == self.min_stack[-1]:
            self.min_stack.pop()

    def top(self):
        return self.stack[-1]

    def get_min(self):
        return self.min_stack[-1]
```

```python
def top_k_largest(nums, k):
    """找最大的k个元素"""
    heap = nums[:k]
    heapq.heapify(heap)       # 先建大小为k的堆

class MedianFinder:
    def __init__(self):
        self.small = []  # 最大堆（存较小的一半，取负）
        self.large = []  # 最小堆（存较大的一半）

    def add_num(self, num):
        # 先加入最大堆
        heapq.heappush(self.small, -num)
        # 把最大堆的最大值移到最小堆
        heapq.heappush(self.large, -heapq.heappop(self.small))
        # 平衡：最大堆的元素数 >= 最小堆
        if len(self.large) > len(self.small):
            heapq.heappush(self.small, -heapq.heappop(self.large))

    def find_median(self):
        if len(self.small) > len(self.large):
            return -self.small[0]
        return (-self.small[0] + self.large[0]) / 2
```

## 13. 贪心算法

### 核心知识点

贪心算法的核心是**每一步都选择局部最优，期望达到全局最优**。

**贪心的关键判断**：贪心选择是否会导致错过全局最优解？如果能证明局部最优一定包含在全局最优中，就可以使用贪心。

### 模板代码

```python
# 模板1：区间贪心（选择不重叠的区间）
def interval_scheduling(intervals):
    """选择尽可能多的不重叠区间"""
    intervals.sort(key=lambda x: x[1])  # 按结束时间排序
    count = 0
    end = float('-inf')
    for start_i, end_i in intervals:
        if start_i >= end:              # 当前区间不与上一个重叠
            count += 1
            end = end_i
    return count

# 模板2：区间合并
def merge_intervals(intervals):
    intervals.sort(key=lambda x: x[0])  # 按开始时间排序
    merged = [intervals[0]]
    for start, end in intervals[1:]:
        if start <= merged[-1][1]:      # 有重叠，合并
            merged[-1][1] = max(merged[-1][1], end)
        else:
            merged.append([start, end])
    return merged

# 模板3：跳跃游戏（贪心维护最远可达位置）
def can_jump(nums):
    max_reach = 0
    for i in range(len(nums)):
        if i > max_reach:
            return False
        max_reach = max(max_reach, i + nums[i])
    return True

# 模板4：买卖股票（贪心：只要有利润就卖）
def max_profit_greedy(prices):
    profit = 0
    for i in range(1, len(prices)):
        if prices[i] > prices[i-1]:
            profit += prices[i] - prices[i-1]  # 收集所有上升段
    return profit
```

```python
# 爬楼梯：dp[i] = dp[i-1] + dp[i-2]
def climb_stairs(n):
    if n <= 2:
        return n
    dp = [0] * (n + 1)
    dp[1] = 1
    dp[2] = 2
    for i in range(3, n + 1):
        dp[i] = dp[i-1] + dp[i-2]

def knapsack_01(weights, values, capacity):
    """
    weights: 物品重量列表
    values: 物品价值列表
    capacity: 背包容量
    每个物品只能选一次
    """
    n = len(weights)
    dp = [0] * (capacity + 1)   # 一维优化
    for i in range(n):
        # 必须倒序遍历！防止同一物品被重复选取
        for w in range(capacity, weights[i] - 1, -1):
            dp[w] = max(dp[w], dp[w - weights[i]] + values[i])
    return dp[capacity]

# ========== 完全背包模板 ==========

def knapsack_complete(weights, values, capacity):
    """
    每个物品可以选无限次
    """
    dp = [0] * (capacity + 1)
    for i in range(len(weights)):
        # 正序遍历！允许同一物品被多次选取
        for w in range(weights[i], capacity + 1):
            dp[w] = max(dp[w], dp[w - weights[i]] + values[i])
    return dp[capacity]

# 最长递增子序列（LIS）
def length_of_lis(nums):
    dp = [1] * len(nums)  # dp[i] = 以nums[i]结尾的LIS长度
    for i in range(1, len(nums)):
        for j in range(i):
            if nums[i] > nums[j]:
                dp[i] = max(dp[i], dp[j] + 1)
    return max(dp)

def length_of_lis_fast(nums):
    tails = []  # tails[i] = 长度为i+1的递增子序列的最小末尾

        pos = bisect.bisect_left(tails, num)
        if pos == len(tails):
            tails.append(num)
        else:
            tails[pos] = num
    return len(tails)

def longest_common_subsequence(text1, text2):
    m, n = len(text1), len(text2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if text1[i-1] == text2[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
    return dp[m][n]

def min_distance(word1, word2):
    m, n = len(word1), len(word2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    # base case
    for i in range(m + 1):
        dp[i][0] = i
    for j in range(n + 1):
        dp[0][j] = j
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if word1[i-1] == word2[j-1]:
                dp[i][j] = dp[i-1][j-1]
            else:
                dp[i][j] = 1 + min(dp[i-1][j],      # 删除
                                   dp[i][j-1],      # 插入
                                   dp[i-1][j-1])    # 替换
    return dp[m][n]
```

```python
def next_permutation(nums):
    """
    找到下一个字典序更大的排列（原地修改）
    步骤：
    1. 从右往左找第一个 nums[i] < nums[i+1] 的位置
    2. 从右往左找第一个 nums[j] > nums[i] 的位置
    3. 交换 nums[i] 和 nums[j]
    4. 反转 nums[i+1:] 使其升序
    """
    n = len(nums)
    i = n - 2
    while i >= 0 and nums[i] >= nums[i + 1]:
        i -= 1
    if i >= 0:
        j = n - 1
        while nums[j] <= nums[i]:
            j -= 1
        nums[i], nums[j] = nums[j], nums[i]
    # 反转 i+1 到末尾
    left, right = i + 1, n - 1
    while left < right:
        nums[left], nums[right] = nums[right], nums[left]
        left += 1
        right -= 1
```

## 分类一：哈希

### 1. [1] 两数之和 (Easy)

**题意简述**：给定一个数组和一个目标值，找出数组中和为目标值的两个数的下标。

**思路分析**：使用哈希表记录已遍历过的元素及其下标。对于每个元素 `num`，检查 `target - num` 是否在哈希表中。

**Python代码**：

```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        seen = {}  # 值 -> 索引
        for i, num in enumerate(nums):
            complement = target - num       # 计算互补值
            if complement in seen:          # 如果互补值已存在
                return [seen[complement], i]
            seen[num] = i                   # 记录当前值
        return []
```

**复杂度分析**：时间 O(n)，空间 O(n)

**要点/易错点**：不能先把所有元素放入哈希表再查找，因为可能出现同一个元素被使用两次的情况。

### 2. [49] 字母异位词分组 (Medium)

**题意简述**：将字母异位词（相同字母不同排列）组合在一起。

**思路分析**：异位词排序后得到相同的字符串，以排序结果作为哈希表的key进行分组。

**Python代码**：

```python
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        groups = defaultdict(list)
        for s in strs:
            key = tuple(sorted(s))    # 排序后作为key（用tuple因为list不可哈希）
            groups[key].append(s)     # 将字符串加入对应分组
        return list(groups.values())
```

**复杂度分析**：时间 O(n * k * log(k))，其中k是字符串最大长度；空间 O(n * k)

**要点/易错点**：key必须是可哈希的类型，所以用 `tuple` 而非 `list`。另一种方法是用字符计数元组（26个字母的频次）作为key。

### 3. [128] 最长连续序列 (Medium)

**题意简述**：给定未排序的整数数组，找出最长连续元素序列的长度。要求 O(n)。

**思路分析**：将数组放入集合中。对于每个数，如果 `num-1` 不在集合中，说明它是一个连续序列的起点，从该起点开始计数。

**Python代码**：

```python
class Solution:
    def longestConsecutive(self, nums: List[int]) -> int:
        num_set = set(nums)         # 放入集合，O(1)查找
        max_len = 0
        for num in num_set:
            # 只有当num是序列起点时才计数
            if num - 1 not in num_set:
                current = num
                length = 1
                while current + 1 in num_set:   # 向后延伸
                    current += 1
                    length += 1
                max_len = max(max_len, length)
        return max_len
```

**复杂度分析**：时间 O(n)，虽然有嵌套循环，但每个元素最多被访问两次；空间 O(n)

**要点/易错点**：必须判断 `num-1` 是否在集合中，否则时间复杂度会退化为 O(n^2)。

```python
class Solution:
```

### 5. [11] 盛最多水的容器 (Medium)

**题意简述**：给定n个非负整数表示宽度为1的柱子高度，找两根柱子使其盛水最多。

**思路分析**：左右对撞指针。每次移动较短的那根柱子（因为移动较长的只会使面积更小或不变）。

**Python代码**：

```python
class Solution:
    def maxArea(self, height: List[int]) -> int:
        left, right = 0, len(height) - 1
        max_water = 0
        while left < right:
            # 当前容器的水量 = 宽度 * 较矮的柱子
            water = (right - left) * min(height[left], height[right])
            max_water = max(max_water, water)
            # 移动较矮的柱子
            if height[left] < height[right]:
                left += 1
            else:
                right -= 1
        return max_water
```

**复杂度分析**：时间 O(n)，空间 O(1)

**要点/易错点**：为什么移动较短的是正确的？因为容器的水量由较短的柱子决定，移动较长的柱子不可能得到更大的水量（宽度在缩小，高度受限于较短的）。

### 6. [15] 三数之和 (Medium)

**题意简述**：找出数组中所有和为0的三元组，不重复。

**思路分析**：排序 + 固定一个数 + 双指针。对于每个 `nums[i]`，在 `nums[i+1..n-1]` 中用对撞指针找两个数使三数之和为0。

**Python代码**：

```python
class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        nums.sort()
        result = []
        for i in range(len(nums) - 2):
            # 去重：跳过相同的nums[i]
            if i > 0 and nums[i] == nums[i - 1]:
                continue
            # 剪枝：最小的三个数之和都大于0
            if nums[i] + nums[i + 1] + nums[i + 2] > 0:
                break
            # 剪枝：当前数加最大的两个数还小于0
            if nums[i] + nums[-1] + nums[-2] < 0:
                continue
            left, right = i + 1, len(nums) - 1
            while left < right:
                s = nums[i] + nums[left] + nums[right]
                if s == 0:
                    result.append([nums[i], nums[left], nums[right]])
                    left += 1
                    right -= 1
                    # 去重
                    while left < right and nums[left] == nums[left - 1]:
                        left += 1
                    while left < right and nums[right] == nums[right + 1]:
                        right -= 1
                elif s < 0:
                    left += 1
                else:
                    right -= 1
        return result
```

**复杂度分析**：时间 O(n^2)，空间 O(log n)（排序）

**要点/易错点**：去重是三数之和的关键难点。需要在三个层面去重：外层循环跳过重复的 `nums[i]`，内层循环跳过重复的 `left` 和 `right`。

### 7. [42] 接雨水 (Hard)

**题意简述**：给定n个非负整数表示柱子高度，计算能接多少雨水。

**思路分析**：
- **方法一（双指针）**：维护左右两侧的最大高度。对于当前位置，能接的水量 = min(左侧最大高度, 右侧最大高度) - 当前高度。
- **方法二（单调栈）**：维护一个单调递减栈，当遇到更高的柱子时，计算凹槽中的水量。

**Python代码**：

```python
# 方法一：左右双指针
class Solution:
    def trap(self, height: List[int]) -> int:
        if not height:
            return 0
        left, right = 0, len(height) - 1
        left_max, right_max = height[left], height[right]
        water = 0
        while left < right:
            if left_max < right_max:
                left += 1
                left_max = max(left_max, height[left])
                water += left_max - height[left]    # 左侧是短板，水量由左侧决定
            else:
                right -= 1
                right_max = max(right_max, height[right])
                water += right_max - height[right]  # 右侧是短板
        return water

# 方法二：单调栈
class Solution2:
    def trap(self, height: List[int]) -> int:
        stack = []  # 存储索引，栈底到栈顶高度递减
        water = 0
        for i in range(len(height)):
            while stack and height[i] > height[stack[-1]]:
                bottom = stack.pop()          # 凹槽底部
                if not stack:
                    break
                # 宽度 = 当前右边界 - 左边界 - 1
                width = i - stack[-1] - 1
                # 高度 = min(左边界高度, 右边界高度) - 底部高度
                h = min(height[stack[-1]], height[i]) - height[bottom]
                water += width * h
            stack.append(i)
        return water
```

**复杂度分析**：两种方法都是时间 O(n)；方法一空间 O(1)，方法二空间 O(n)

**要点/易错点**：接雨水是经典Hard题，关键理解是"每个位置能接多少水取决于它两侧最高柱子中较矮的那个"。单调栈方法中，注意计算的是横向的水量。

### 8. [3] 无重复字符的最长子串 (Medium)

**题意简述**：找出不含重复字符的最长子串的长度。

**思路分析**：滑动窗口 + 哈希集合。扩大窗口加入新字符，当出现重复时缩小窗口直到无重复。

**Python代码**：

```python
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        char_set = set()    # 记录窗口内的字符
        left = 0
        max_len = 0
        for right in range(len(s)):
            while s[right] in char_set:          # 出现重复，缩小窗口
                char_set.remove(s[left])
                left += 1
            char_set.add(s[right])               # 加入新字符
            max_len = max(max_len, right - left + 1)
        return max_len
```

**复杂度分析**：时间 O(n)，每个字符最多进出窗口各一次；空间 O(min(n, m))，m为字符集大小

### 9. [438] 找到字符串中所有字母异位词 (Medium)

**题意简述**：找到字符串s中所有是p的字母异位词的子串，返回起始索引列表。

**思路分析**：定长滑动窗口（窗口长度 = len(p)）。维护窗口内字符计数与p的字符计数的匹配情况。

**Python代码**：

```python
class Solution:
    def findAnagrams(self, s: str, p: str) -> List[int]:
        if len(s) < len(p):
            return []
        p_count = Counter(p)
        window_count = Counter(s[:len(p)])    # 初始窗口
        result = []
        if window_count == p_count:
            result.append(0)
        for i in range(len(p), len(s)):
            # 滑入新字符
            window_count[s[i]] += 1
            # 滑出旧字符
            window_count[s[i - len(p)]] -= 1
            if window_count[s[i - len(p)]] == 0:
                del window_count[s[i - len(p)]]  # 清理0值
            # 检查是否匹配
            if window_count == p_count:
                result.append(i - len(p) + 1)
        return result
```

**复杂度分析**：时间 O(n)，空间 O(m)，m为字符集大小

### 10. [560] 和为K的子数组 (Medium)

**题意简述**：找出数组中和为k的连续子数组的个数。

**思路分析**：前缀和 + 哈希表。如果 `prefix[j] - prefix[i] = k`，则 `nums[i+1..j]` 的和为k。

**Python代码**：

```python

        prefix_count[0] = 1          # 前缀和为0出现1次
        current_sum = 0
        count = 0
        for num in nums:
            current_sum += num
            count += prefix_count[current_sum - k]   # 查找满足条件的前缀和
            prefix_count[current_sum] += 1           # 记录当前前缀和
        return count
```

**复杂度分析**：时间 O(n)，空间 O(n)

**要点/易错点**：注意 `prefix_count[0] = 1` 的初始化，它表示空前缀的和为0。

### 11. [239] 滑动窗口最大值 (Hard)

**题意简述**：给定数组和窗口大小k，返回每个窗口的最大值。

**思路分析**：单调队列（单调递减的双端队列）。队列中存储索引，保证队首始终是当前窗口的最大值。

**Python代码**：

```python
class Solution:
    def maxSlidingWindow(self, nums: List[int], k: int) -> List[int]:
        from collections import deque
        dq = deque()     # 存储索引，对应的值单调递减
        result = []
        for i in range(len(nums)):
            # 移除窗口外的元素
            while dq and dq[0] < i - k + 1:
                dq.popleft()
            # 维护单调性：移除比当前元素小的
            while dq and nums[dq[-1]] <= nums[i]:
                dq.pop()
            dq.append(i)
            # 窗口形成后记录结果
            if i >= k - 1:
                result.append(nums[dq[0]])
        return result
```

**复杂度分析**：时间 O(n)，每个元素最多入队出队各一次；空间 O(k)

### 12. [76] 最小覆盖子串 (Hard)

**题意简述**：找到字符串s中包含字符串t所有字符的最小子串。

**思路分析**：不定长滑动窗口。扩大窗口直到包含t的所有字符，然后缩小窗口寻找最小覆盖子串。

**Python代码**：

```python
class Solution:
    def minWindow(self, s: str, t: str) -> str:
        need = Counter(t)           # t中每个字符的需求量
        window = defaultdict(int)   # 当前窗口中的字符计数
        have = 0                    # 已满足的字符种类数
        required = len(need)        # 需要满足的字符种类数
        left = 0
        min_len = float('inf')
        min_start = 0

        for right in range(len(s)):
            char = s[right]
            window[char] += 1
            # 当某个字符的数量达到需求，满足种类+1
            if char in need and window[char] == need[char]:
                have += 1

            # 当所有字符都满足时，尝试缩小窗口
            while have == required:
                # 更新最小窗口
                if right - left + 1 < min_len:
                    min_len = right - left + 1
                    min_start = left
                # 缩小窗口
                left_char = s[left]
                window[left_char] -= 1
                if left_char in need and window[left_char] < need[left_char]:
                    have -= 1          # 不再满足该字符的需求
                left += 1

        return s[min_start:min_start + min_len] if min_len != float('inf') else ""
```

**复杂度分析**：时间 O(n)，空间 O(m)，m为字符集大小

**要点/易错点**：用 `have` 和 `required` 来跟踪满足情况，避免每次都遍历哈希表比较。注意只在窗口缩小时更新答案。

```python
        n = len(nums)
        answer = [1] * n

        # 计算前缀积（左侧所有元素的乘积）
        prefix = 1
        for i in range(n):
            answer[i] = prefix
            prefix *= nums[i]

        # 计算后缀积（右侧所有元素的乘积），同时合并到answer中
        suffix = 1
        for i in range(n - 1, -1, -1):
            answer[i] *= suffix
            suffix *= nums[i]

        return answer
```

**Python代码**：

```python

    def firstMissingPositive(self, nums: List[int]) -> int:
        n = len(nums)
        # 将每个正整数放到正确位置
        for i in range(n):
            while 1 <= nums[i] <= n and nums[nums[i] - 1] != nums[i]:

                nums[i], nums[correct_idx] = nums[correct_idx], nums[i]

        for i in range(n):
            if nums[i] != i + 1:
                return i + 1
        return n + 1  # 所有位置都正确，缺失的是 n+1
```

```python

        dummy.next = head
        prev_group_end = dummy

        while True:
            # 找到当前组的尾节点
            group_end = prev_group_end
            for _ in range(k):
                group_end = group_end.next
                if not group_end:
                    return dummy.next  # 不足k个，结束
            # 记录下一组的起始节点

            # 翻转当前组
            group_start = prev_group_end.next
            prev = next_group_start
            curr = group_start
            while curr != next_group_start:
                nxt = curr.next
                curr.next = prev
                prev = curr
                curr = nxt
            # 连接
            prev_group_end.next = prev
            prev_group_end = group_start

        return dummy.next
```

### 35. [146] LRU缓存 (Medium)

**题意简述**：设计一个LRU缓存，支持O(1)的get和put操作。

**思路分析**：哈希表 + 双向链表。

**Python代码**：

```python
class DLinkedNode:
    def __init__(self, key=0, val=0):
        self.key = key
        self.val = val
        self.prev = None
        self.next = None

class LRUCache:
    def __init__(self, capacity: int):
        self.cap = capacity
        self.cache = {}
        self.head = DLinkedNode()
        self.tail = DLinkedNode()
        self.head.next = self.tail
        self.tail.prev = self.head

    def _remove(self, node):
        node.prev.next = node.next
        node.next.prev = node.prev

    def _add_to_head(self, node):
        node.prev = self.head
        node.next = self.head.next
        self.head.next.prev = node
        self.head.next = node

    def _move_to_head(self, node):
        self._remove(node)
        self._add_to_head(node)

    def get(self, key: int) -> int:
        if key not in self.cache:
            return -1
        node = self.cache[key]
        self._move_to_head(node)
        return node.val

    def put(self, key: int, value: int) -> None:
        if key in self.cache:
            node = self.cache[key]
            node.val = value
            self._move_to_head(node)
        else:
            node = DLinkedNode(key, value)
            self.cache[key] = node
            self._add_to_head(node)
            if len(self.cache) > self.cap:
                removed = self.tail.prev
                self._remove(removed)
                del self.cache[removed.key]
```

**复杂度分析**：get和put均为时间 O(1)，空间 O(capacity)

**要点/易错点**：节点必须存储key，因为在淘汰时需要从哈希表中删除对应的key。

### 62. [51] N皇后 (Hard)

**思路分析**：逐行放置皇后，用集合记录已被占用的列和对角线。

```python
class Solution:
    def solveNQueens(self, n: int) -> List[List[str]]:
        result = []
        board = [['.'] * n for _ in range(n)]
        cols = set()
        diag1 = set()       # row - col 相同
        diag2 = set()       # row + col 相同

        def backtrack(row):
            if row == n:
                result.append([''.join(r) for r in board])
                return
            for col in range(n):
                if col in cols or (row - col) in diag1 or (row + col) in diag2:
                    continue
                board[row][col] = 'Q'
                cols.add(col)
                diag1.add(row - col)
                diag2.add(row + col)
                backtrack(row + 1)
                board[row][col] = '.'
                cols.remove(col)
                diag1.remove(row - col)
                diag2.remove(row + col)

        backtrack(0)
        return result
```

**复杂度分析**：时间 O(n!)，空间 O(n)

**要点/易错点**：主对角线上 `row - col` 为常数，副对角线上 `row + col` 为常数。

```python
                dp[j] = dp[j] or dp[j - num]
        return dp[target]
```

### 94. [5] 最长回文子串 (Medium)

**思路分析**：DP或中心扩展。`dp[i][j]` 表示 `s[i..j]` 是否为回文。

```python
# 方法一：中心扩展（更简洁）
class Solution:
    def longestPalindrome(self, s: str) -> str:
        def expand(left, right):
            while left >= 0 and right < len(s) and s[left] == s[right]:
                left -= 1
                right += 1
            return s[left + 1:right]

        result = ''
        for i in range(len(s)):
            # 奇数长度
            odd = expand(i, i)
            if len(odd) > len(result):
                result = odd
            # 偶数长度
            even = expand(i, i + 1)
            if len(even) > len(result):
                result = even
        return result

# 方法二：DP
class Solution2:
    def longestPalindrome(self, s: str) -> str:
        n = len(s)
        dp = [[False] * n for _ in range(n)]
        start, max_len = 0, 1
        for i in range(n):
            dp[i][i] = True
        for length in range(2, n + 1):
            for i in range(n - length + 1):
                j = i + length - 1
                if s[i] == s[j]:
                    if length <= 3:
                        dp[i][j] = True
                    else:
                        dp[i][j] = dp[i+1][j-1]
                if dp[i][j] and length > max_len:
                    start, max_len = i, length
        return s[start:start + max_len]
```

**复杂度分析**：中心扩展时间 O(n^2)，空间 O(1)；DP时间 O(n^2)，空间 O(n^2)

### 98. [132] 分割回文串 II (Hard)

**题意简述**：将字符串分割成最少数量的回文子串，返回最少分割次数。

**思路分析**：先用DP预处理出所有子串是否为回文，再用DP求最少分割次数。

```python
class Solution:
    def minCut(self, s: str) -> int:
        n = len(s)
        # 预处理回文
        is_pal = [[False] * n for _ in range(n)]
        for i in range(n):
            is_pal[i][i] = True
        for length in range(2, n + 1):
            for i in range(n - length + 1):
                j = i + length - 1
                if s[i] == s[j]:
                    if length <= 3:
                        is_pal[i][j] = True
                    else:
                        is_pal[i][j] = is_pal[i+1][j-1]

        # dp[i] = s[0..i]的最少分割次数
        dp = list(range(n))  # 最坏情况：每个字符一刀
        for i in range(1, n):
            if is_pal[0][i]:
                dp[i] = 0
                continue
            for j in range(1, i + 1):
                if is_pal[j][i]:
                    dp[i] = min(dp[i], dp[j-1] + 1)
        return dp[n-1]
```

**复杂度分析**：时间 O(n^2)，空间 O(n^2)

### 99. [10] 正则表达式匹配 (Hard)

**题意简述**：实现支持 `.` 和 `*` 的正则表达式匹配。`.` 匹配任意字符，`*` 匹配零个或多个前面的元素。

**思路分析**：二维DP。`dp[i][j]` 表示 `s[0..i-1]` 和 `p[0..j-1]` 是否匹配。

```python
class Solution:
    def isMatch(self, s: str, p: str) -> bool:
        m, n = len(s), len(p)
        dp = [[False] * (n + 1) for _ in range(m + 1)]
        dp[0][0] = True  # 空串匹配空模式

        # 处理 p 的前缀 * 可以匹配空串的情况
        for j in range(2, n + 1):
            if p[j-1] == '*':
                dp[0][j] = dp[0][j-2]

        for i in range(1, m + 1):
            for j in range(1, n + 1):
                if p[j-1] == '*':
                    # * 匹配0个前面的字符
                    dp[i][j] = dp[i][j-2]
                    # * 匹配1个或多个：如果当前字符匹配
                    if p[j-2] == '.' or p[j-2] == s[i-1]:
                        dp[i][j] = dp[i][j] or dp[i-1][j]
                elif p[j-1] == '.' or p[j-1] == s[i-1]:
                    dp[i][j] = dp[i-1][j-1]

        return dp[m][n]
```

**复杂度分析**：时间 O(m*n)，空间 O(m*n)

**要点/易错点**：`*` 的处理是难点。当 `*` 匹配0个时看 `dp[i][j-2]`；匹配多个时看 `dp[i-1][j]`（前提是当前字符匹配）。

### 100. [312] 戳气球 (Hard)

**题意简述**：戳破气球i获得 `nums[i-1] * nums[i] * nums[i+1]` 的金币，求最大金币数。

**思路分析**：区间DP。逆向思考：不是戳破气球，而是最后戳破哪个气球。`dp[i][j]` 表示戳破 `(i, j)` 开区间内所有气球的最大金币数。

```python
class Solution:
    def maxCoins(self, nums: List[int]) -> int:
        # 在两端添加虚拟气球（值为1）
        nums = [1] + nums + [1]
        n = len(nums)
        dp = [[0] * n for _ in range(n)]

        # 区间长度从2开始（至少包含一个气球）
        for length in range(2, n):
            for i in range(n - length):
                j = i + length
                # 枚举(i, j)区间内最后戳破的气球k
                for k in range(i + 1, j):
                    dp[i][j] = max(dp[i][j],
                                   dp[i][k] + dp[k][j] + nums[i] * nums[k] * nums[j])
        return dp[0][n-1]
```

**复杂度分析**：时间 O(n^3)，空间 O(n^2)

**要点/易错点**：关键在于"逆向思维"——考虑最后戳破哪个气球而非第一个。在两端添加虚拟气球简化边界处理。

### 105. [166] 分数到小数 (Medium)

**思路分析**：模拟长除法，用哈希表记录余数出现的位置来检测循环节。

```python
class Solution:
    def fractionToDecimal(self, numerator: int, denominator: int) -> str:
        # 处理符号
        if numerator * denominator < 0:
            sign = '-'
        else:
            sign = ''
        numerator, denominator = abs(numerator), abs(denominator)

        # 整数部分
        integer_part = numerator // denominator
        remainder = numerator % denominator

        if remainder == 0:
            return sign + str(integer_part)

        # 小数部分
        decimal = []
        remainder_map = {}
        while remainder != 0:
            if remainder in remainder_map:
                # 发现循环节
                idx = remainder_map[remainder]
                decimal.insert(idx, '(')
                decimal.append(')')
                break
            remainder_map[remainder] = len(decimal)
            remainder *= 10
            decimal.append(str(remainder // denominator))
            remainder %= denominator

        return sign + str(integer_part) + '.' + ''.join(decimal)
```

**复杂度分析**：时间 O(d)，d为小数位数；空间 O(d)

### 106. [149] 直线上最多的点数 (Hard)

**题意简述**：给定平面上的点，找出同一直线上最多的点数。

**思路分析**：对每个点，计算它与其他所有点的斜率，用哈希表统计同一斜率的点数。

```python
class Solution:
    def maxPoints(self, points: List[List[int]]) -> int:
        from math import gcd
        if len(points) <= 2:
            return len(points)

        max_count = 0
        for i in range(len(points)):
            slope_count = defaultdict(int)
            for j in range(i + 1, len(points)):
                dx = points[j][0] - points[i][0]
                dy = points[j][1] - points[i][1]
                # 化简斜率为最简分数
                g = gcd(dx, dy)
                slope = (dx // g, dy // g)
                slope_count[slope] += 1
                max_count = max(max_count, slope_count[slope])
        return max_count + 1  # +1 是加上当前点本身
```

**复杂度分析**：时间 O(n^2)，空间 O(n)

### 按算法模式分类

**模式一：遍历 + 哈希**（入门必会）
- 两数之和、字母异位词分组、最长连续序列

**模式二：排序 + 双指针**（数组三数之和系列）
- 盛水容器、三数之和、颜色分类

**模式三：滑动窗口**（子串问题万能模板）
- 无重复最长子串、字母异位词、最小覆盖子串

**模式四：递归 + 分治**（树的灵魂）
- 最大深度、翻转二叉树、对称二叉树、LCA、直径

**模式五：DFS/BFS**（图和矩阵问题）
- 岛屿数量、腐烂的橘子、课程表

**模式六：回溯**（组合排列问题）
- 全排列、子集、组合总和、N皇后

**模式七：二分查找**（有序数组搜索）
- 搜索旋转数组、搜索二维矩阵、寻找中位数

**模式八：一维DP**（序列问题）
- 爬楼梯、打家劫舍、LIS、乘积最大子数组

**模式九：二维DP**（网格/字符串匹配）
- 不同路径、编辑距离、LCS、最长回文子串

**模式十：设计类**（综合数据结构）
- LRU缓存、最小栈、Trie、中位数查找器

### 每道题的做题流程

1. **独立思考15分钟**：先看题意，想思路
2. **看提示/模板**：如果没思路，参考本文档的模板部分

### 常见错误提醒

- **忘记处理边界条件**：空数组、空链表、空树
- **整数溢出**：Python不会溢出，但面试其他语言时要注意
- **浅拷贝问题**：回溯中 `result.append(path[:])` 不能写成 `result.append(path)`
- **死循环**：链表操作中忘记更新指针、二分查找边界更新错误
- **去重遗漏**：三数之和、全排列去重等场景
